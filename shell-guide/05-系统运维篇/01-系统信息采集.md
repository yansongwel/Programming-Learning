# 系统信息采集

> 一句话说明：用 Shell 脚本全面采集服务器硬件、操作系统和运行状态信息。

---

## 1. 基本系统信息

```bash
# 主机名
hostname
hostnamectl

# 操作系统
cat /etc/os-release
uname -a                         # 内核信息
uname -r                         # 内核版本

# 运行时间
uptime
uptime -p                        # 人性化显示

# 系统负载
cat /proc/loadavg                # 1/5/15 分钟负载
```

---

## 2. CPU 信息

```bash
# CPU 型号和核心数
lscpu
nproc                            # 逻辑 CPU 数
grep "model name" /proc/cpuinfo | head -1
grep -c processor /proc/cpuinfo  # CPU 核心数

# CPU 使用率
top -bn1 | grep "Cpu(s)" | awk '{print "使用率:", 100-$8"%"}'

# 每核 CPU 使用率
mpstat -P ALL 1 1 2>/dev/null || echo "需安装 sysstat"
```

---

## 3. 内存信息

```bash
# 内存概览
free -h
free -m

# 详细信息
cat /proc/meminfo | head -10

# 使用率计算
free | awk '/Mem:/{printf "内存使用率: %.1f%%\n", $3/$2*100}'

# 谁在用内存
ps aux --sort=-%mem | head -10
smem -tk 2>/dev/null || echo "可安装 smem 查看详细内存"
```

---

## 4. 磁盘信息

```bash
# 分区和使用情况
df -hT                           # 带文件系统类型
lsblk                            # 块设备树

# inode 使用
df -i

# 磁盘 I/O
iostat -xz 1 3 2>/dev/null || echo "需安装 sysstat"

# 各目录大小
du -sh /var/* 2>/dev/null | sort -rh | head -10
```

---

## 5. 网络信息

```bash
# IP 地址
ip -4 addr show | grep inet | awk '{print $NF": "$2}'

# 路由
ip route show | head -5

# 监听端口
ss -tlnp

# 连接统计
ss -s
```

## 6. 实战：服务器巡检脚本

```bash
#!/usr/bin/env bash
# server_inspect.sh — 一键采集服务器信息

set -euo pipefail

SEP="$(printf '=%.0s' {1..60})"

section() { echo ""; echo "$SEP"; echo "  $1"; echo "$SEP"; }

section "基本信息"
echo "主机名:   $(hostname)"
echo "IP地址:   $(hostname -I | awk '{print $1}')"
echo "系统:     $(grep PRETTY_NAME /etc/os-release 2>/dev/null | cut -d'"' -f2 || uname -s)"
echo "内核:     $(uname -r)"
echo "运行时间: $(uptime -p 2>/dev/null || uptime)"
echo "当前时间: $(date '+%Y-%m-%d %H:%M:%S %Z')"

section "CPU"
echo "型号: $(grep 'model name' /proc/cpuinfo | head -1 | cut -d: -f2 | xargs)"
echo "核心: $(nproc) 核"
echo "负载: $(cat /proc/loadavg | awk '{print $1, $2, $3}')"

section "内存"
free -h | awk '
    /Mem:/{printf "物理内存: %s / %s (使用率 %.1f%%)\n", $3, $2, $3/$2*100}
    /Swap:/{if($2!="0B" && $2!="0") printf "Swap:     %s / %s\n", $3, $2}
'

section "磁盘"
printf "%-25s %8s %8s %8s %6s\n" "挂载点" "总量" "已用" "可用" "使用率"
df -h | awk 'NR>1 && $1!~/tmpfs|devtmpfs/{printf "%-25s %8s %8s %8s %6s\n", $6, $2, $3, $4, $5}'

section "网络"
echo "监听端口:"
ss -tlnp 2>/dev/null | awk 'NR>1{printf "  %-6s %s\n", $4, $NF}' | head -20

section "Top 进程 (内存)"
printf "  %-8s %6s %6s  %s\n" "PID" "CPU%" "MEM%" "CMD"
ps aux --sort=-%mem | awk 'NR>1 && NR<=6{printf "  %-8s %6s %6s  %s\n", $2, $3, $4, $11}'
```

| 内存 | `free -h`, `/proc/meminfo` |
| 磁盘 | `df -hT`, `lsblk`, `du -sh` |
