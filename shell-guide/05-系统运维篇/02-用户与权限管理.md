# 用户与权限管理

> 一句话说明：掌握 Linux 用户管理和文件权限体系，保障系统安全。

---

## 1. 用户管理

```bash
# 查看用户
whoami                           # 当前用户
id                               # UID/GID 信息
id username                      # 指定用户信息
cat /etc/passwd                  # 所有用户
getent passwd                    # 包含 LDAP 等外部源

# 创建用户
useradd -m -s /bin/bash username          # 创建用户+主目录
useradd -m -G wheel,docker username       # 指定附加组
echo "username:password" | chpasswd       # 设置密码

# 修改用户
usermod -aG docker username       # 添加到组（-a 追加）
usermod -s /bin/zsh username      # 修改 Shell
usermod -L username               # 锁定用户
usermod -U username               # 解锁用户

# 删除用户
userdel username                  # 删除用户
userdel -r username               # 删除用户+主目录

# 密码管理
passwd username                   # 修改密码
chage -l username                 # 查看密码策略
chage -M 90 username              # 密码 90 天过期
```

## 2. 组管理

```bash
# 查看组
groups username
cat /etc/group

# 管理组
groupadd developers
groupmod -n newname oldname
groupdel groupname
gpasswd -a user group             # 添加用户到组
gpasswd -d user group             # 从组中移除用户
```

## 3. 文件权限

```bash
# 权限表示：rwxrwxrwx = 所有者|组|其他
# r=4  w=2  x=1

# 查看权限
ls -la file

# chmod — 修改权限
chmod 755 file                    # rwxr-xr-x
chmod 644 file                    # rw-r--r--
chmod +x script.sh                # 添加执行权限
chmod -R 755 dir/                 # 递归

# chown — 修改属主
chown user:group file
chown -R user:group dir/

# 特殊权限
chmod u+s file                    # SUID
chmod g+s dir                     # SGID
chmod +t dir                      # Sticky bit（如 /tmp）

# umask — 默认权限掩码
umask                             # 查看当前值
umask 022                         # 文件默认 644，目录默认 755
```

## 5. 实战：批量用户管理

```bash
#!/usr/bin/env bash
# user_manager.sh — 批量用户管理

set -euo pipefail

ACTION="${1:?用法: $0 <create|delete|lock|list> [用户列表文件]}"

case "$ACTION" in
    create)
        FILE="${2:?请指定用户列表文件}"
        while IFS=: read -r user groups comment; do
            [[ -z "$user" || "$user" =~ ^# ]] && continue
            if id "$user" &>/dev/null; then
                echo "跳过: $user (已存在)"
                continue
            fi
            useradd -m -c "$comment" ${groups:+-G "$groups"} "$user"
            # 生成随机密码
            PASS=$(openssl rand -base64 12)
            echo "$user:$PASS" | chpasswd
            chage -d 0 "$user"      # 首次登录强制改密
            echo "已创建: $user (密码: $PASS)"
        done < "$FILE"
        ;;
    lock)
        FILE="${2:?请指定用户列表文件}"
        while read -r user; do
            [[ -z "$user" || "$user" =~ ^# ]] && continue
            usermod -L "$user" && echo "已锁定: $user"
        done < "$FILE"
        ;;
    list)
        echo "系统用户列表 (UID >= 1000):"
        awk -F: '$3>=1000 && $3<65534{printf "  %-15s UID:%-6d %s\n", $1, $3, $7}' /etc/passwd
        ;;
    *)
        echo "未知操作: $ACTION"
        exit 1
        ;;
esac
```

## 6. 小结

| 操作 | 命令 |
|------|------|
| 用户创建 | `useradd -m -s /bin/bash` |
| 权限修改 | `chmod 755`, `chown user:group` |
| 组管理 | `groupadd`, `usermod -aG` |
| sudo | `visudo` 编辑配置 |
| 安全 | `chage` 密码策略, `usermod -L` 锁定 |
