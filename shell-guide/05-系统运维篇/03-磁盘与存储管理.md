# 磁盘与存储管理

> 一句话说明：掌握磁盘分区、文件系统、LVM 和磁盘监控，保障存储系统可靠运行。

---

## 1. 磁盘信息查看

```bash
# 块设备列表
lsblk
lsblk -f                         # 显示文件系统类型

# 磁盘详细信息
fdisk -l                         # 分区表
blkid                            # UUID 和文件系统

# 使用情况
df -hT                           # 分区使用
df -i                            # inode 使用
du -sh /var/*  | sort -rh        # 目录空间
```

---

## 2. 分区管理

```bash
# fdisk — MBR 分区（< 2TB）
fdisk /dev/sdb
# n: 新建分区  d: 删除  p: 打印  w: 写入  q: 退出

# gdisk / parted — GPT 分区（> 2TB）
parted /dev/sdb
# mklabel gpt
# mkpart primary ext4 0% 100%

# 格式化
mkfs.ext4 /dev/sdb1
mkfs.xfs /dev/sdb1

# 挂载
mount /dev/sdb1 /data
umount /data

# 永久挂载 (/etc/fstab)
echo "UUID=$(blkid -s UUID -o value /dev/sdb1)  /data  ext4  defaults  0  2" >> /etc/fstab
mount -a                          # 验证 fstab
```

## 3. LVM 管理

```bash
# PV → VG → LV
# 物理卷 → 卷组 → 逻辑卷

# 创建
pvcreate /dev/sdb /dev/sdc
vgcreate data_vg /dev/sdb /dev/sdc
lvcreate -n data_lv -L 100G data_vg
mkfs.ext4 /dev/data_vg/data_lv
mount /dev/data_vg/data_lv /data

# 扩容（LVM 的核心优势）
lvextend -L +50G /dev/data_vg/data_lv
resize2fs /dev/data_vg/data_lv      # ext4
xfs_growfs /data                     # xfs

# 一步扩容
lvextend -r -L +50G /dev/data_vg/data_lv   # -r 自动扩展文件系统

# 查看
pvs; vgs; lvs
pvdisplay; vgdisplay; lvdisplay
```

## 4. 磁盘监控脚本

```bash
#!/usr/bin/env bash
# disk_monitor.sh — 磁盘空间监控与告警

set -euo pipefail

WARN_THRESHOLD="${1:-80}"
CRIT_THRESHOLD="${2:-90}"

echo "磁盘空间检查 (警告: ${WARN_THRESHOLD}% / 严重: ${CRIT_THRESHOLD}%)"
echo "================================================"

STATUS="OK"
while read -r usage mount fstype; do
    usage_num=${usage%\%}
    if (( usage_num >= CRIT_THRESHOLD )); then
        printf "  [严重] %-20s %s (%s)\n" "$mount" "$usage" "$fstype"
        STATUS="CRITICAL"
    elif (( usage_num >= WARN_THRESHOLD )); then
        printf "  [警告] %-20s %s (%s)\n" "$mount" "$usage" "$fstype"
        [[ "$STATUS" != "CRITICAL" ]] && STATUS="WARNING"
    else
        printf "  [正常] %-20s %s (%s)\n" "$mount" "$usage" "$fstype"
    fi
done < <(df -hT | awk 'NR>1 && $1!~/tmpfs|devtmpfs/{print $6, $7, $2}')

echo ""
echo "整体状态: $STATUS"

# inode 检查
echo ""
echo "inode 使用情况:"
df -i | awk 'NR>1 && $1!~/tmpfs/ && $5!="-"{
    pct=+$5
    if(pct>80) printf "  [警告] %-20s inode使用: %s\n", $6, $5
}'
```

## 5. 磁盘清理

```bash
# 查找大文件
find / -type f -size +100M -exec ls -lh {} \; 2>/dev/null | sort -k5 -rh | head -20

# 清理日志
journalctl --vacuum-size=100M     # 清理 journal 日志
find /var/log -name "*.gz" -mtime +30 -delete

# 清理软件包缓存
apt clean       # Debian/Ubuntu
yum clean all   # CentOS/RHEL

# 查找并清理删除但未释放的文件
lsof +L1 | grep deleted
```

## 6. 小结

|------|------|

| LVM | `pvcreate→vgcreate→lvcreate→lvextend` |
