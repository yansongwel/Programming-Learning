# 文件与目录操作

> 一句话说明：掌握 Linux 文件系统的核心命令，高效管理文件和目录。

---

## 1. 文件基本操作

### 1.1 创建

```bash
# 创建文件
touch file.txt                   # 创建空文件（或更新时间戳）
> file.txt                       # 创建空文件（清空已有内容）

# 创建目录
mkdir dir                        # 创建单级目录
mkdir -p a/b/c/d                 # 递归创建多级目录
mkdir -m 755 mydir               # 创建时指定权限

# 创建临时文件/目录
TMPFILE=$(mktemp)                # /tmp/tmp.XXXXXXXXXX
TMPDIR=$(mktemp -d)              # 创建临时目录
```

### 1.2 复制

```bash
cp file1 file2                   # 复制文件
cp -r dir1 dir2                  # 递归复制目录
cp -a dir1 dir2                  # 保留所有属性（归档模式）
cp -v file1 file2                # 显示复制过程
cp -n file1 file2                # 不覆盖已存在的文件
cp -u file1 file2                # 只在源文件更新时才复制
cp --backup=numbered f1 f2       # 覆盖时自动备份
```

### 1.3 移动/重命名

```bash
mv file1 file2                   # 重命名
mv file1 /path/to/dir/           # 移动
mv -n file1 file2                # 不覆盖
mv -v file1 file2                # 显示过程

# 批量重命名
for f in *.txt; do
    mv "$f" "${f%.txt}.md"
done

# rename 命令（Perl 版本）
rename 's/\.txt$/.md/' *.txt
```

### 1.4 删除

```bash
rm file                          # 删除文件
rm -r dir                        # 递归删除目录
rm -f file                       # 强制删除（不提示）
rm -rf dir                       # 强制递归删除（危险！）

# 安全删除习惯
rm -i file                       # 交互确认
rm -v file                       # 显示过程
# 生产环境建议先 mv 到回收站目录
```

## 2. 查找文件 — find

```bash
# 按名称
find /var/log -name "*.log"
find . -iname "readme*"            # 忽略大小写

# 按类型
find . -type f                     # 普通文件
find . -type d                     # 目录
find . -type l                     # 符号链接

# 按大小
find / -size +100M                 # 大于 100MB
find . -size -1k                   # 小于 1KB
find . -size 0                     # 空文件

# 按时间
find . -mtime -7                   # 7 天内修改过
find . -mtime +30                  # 30 天前修改的
find . -newer reference.txt        # 比 reference.txt 新

# 按权限
find . -perm 755                   # 精确匹配
find . -perm -u=x                  # 用户可执行

# 组合条件
find /var/log -name "*.log" -size +10M -mtime +30

# 执行操作
find . -name "*.tmp" -delete                           # 删除
find . -name "*.sh" -exec chmod +x {} \;               # 逐个执行
find . -name "*.log" -exec gzip {} \;                  # 逐个压缩
find . -name "*.bak" -exec rm {} +                     # 批量执行（更高效）
find . -name "*.conf" -exec grep -l "old" {} \;        # 查找包含内容的文件

# 实用组合
# 清理 30 天前的日志
find /var/log -name "*.log" -mtime +30 -delete

# 查找大文件 Top 10
find / -type f -exec du -h {} + 2>/dev/null | sort -rh | head -10

# 查找空目录
find . -type d -empty
```

## 3. 文件内容查看

```bash
cat file                         # 输出全部内容
cat -n file                      # 显示行号
cat -A file                      # 显示特殊字符

head -20 file                    # 前 20 行
tail -20 file                    # 后 20 行
tail -f file                     # 实时跟踪（日志神器）
tail -f -n 100 app.log           # 显示最后 100 行并跟踪

less file                        # 分页查看（推荐）
# less 快捷键: /搜索 n下一个 N上一个 g首行 G末行 q退出

wc file                          # 行数 词数 字节数
wc -l file                       # 只统计行数

file myfile                      # 查看文件类型
stat myfile                      # 详细文件信息
```

## 4. 链接

```bash
# 硬链接（指向同一 inode）
ln file1 file2
# 特点：删除原文件不影响硬链接，不能跨文件系统，不能链接目录

# 软链接/符号链接（类似快捷方式）
ln -s /path/to/target linkname
# 特点：删除原文件则链接失效，可跨文件系统，可链接目录

# 查看链接
ls -la linkname
readlink -f linkname             # 获取链接的真实路径

# 实用场景
ln -sf /opt/app/v2.0 /opt/app/current    # 版本切换
```

## 5. 压缩与归档

```bash
# tar — 打包归档
tar cf archive.tar dir/           # 打包
tar czf archive.tar.gz dir/       # 打包 + gzip 压缩
tar cjf archive.tar.bz2 dir/      # 打包 + bzip2 压缩
tar xf archive.tar.gz              # 解压
tar xf archive.tar.gz -C /dest/   # 解压到指定目录
tar tf archive.tar.gz              # 查看内容

# gzip / gunzip
gzip file                         # 压缩（替换原文件）
gzip -k file                      # 压缩（保留原文件）
gunzip file.gz                    # 解压

# zip / unzip
zip -r archive.zip dir/
unzip archive.zip
unzip -l archive.zip              # 查看内容

# 实用：打包时排除文件
tar czf backup.tar.gz --exclude='*.log' --exclude='node_modules' ./project/
```

## 6. 磁盘使用

```bash
# df — 文件系统使用情况
df -h                            # 人类可读
df -hT                           # 带文件系统类型
df -i                            # inode 使用

# du — 目录空间占用
du -sh dir/                      # 目录总大小
du -sh *                         # 当前目录下各项大小
du -sh * | sort -rh | head -10   # 最大的 10 个

# ncdu — 交互式磁盘分析（推荐安装）
ncdu /var/log
```

## 7. 实战练习

### 练习：自动备份脚本

```bash
#!/usr/bin/env bash
# backup.sh — 每日备份指定目录

set -euo pipefail

SOURCE="${1:?用法: $0 <源目录> [备份目录]}"
BACKUP_DIR="${2:-/backup}"
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_NAME="$(basename "$SOURCE")_${DATE}.tar.gz"
KEEP_DAYS=7

mkdir -p "$BACKUP_DIR"

echo "开始备份: $SOURCE → $BACKUP_DIR/$BACKUP_NAME"
tar czf "$BACKUP_DIR/$BACKUP_NAME" -C "$(dirname "$SOURCE")" "$(basename "$SOURCE")"

SIZE=$(du -h "$BACKUP_DIR/$BACKUP_NAME" | cut -f1)
echo "备份完成: $SIZE"

# 清理旧备份
OLD_COUNT=$(find "$BACKUP_DIR" -name "$(basename "$SOURCE")_*.tar.gz" -mtime +$KEEP_DAYS | wc -l)
if (( OLD_COUNT > 0 )); then
    find "$BACKUP_DIR" -name "$(basename "$SOURCE")_*.tar.gz" -mtime +$KEEP_DAYS -delete
    echo "已清理 $OLD_COUNT 个过期备份"
fi
```

## 8. 小结

| 操作 | 常用命令 |
|------|----------|
| 查找文件 | `find` + `-name/-size/-mtime/-exec` |
| 文件内容 | `cat/head/tail/less/wc` |
| 压缩归档 | `tar czf`(压缩) `tar xf`(解压) |
| 磁盘分析 | `df -h`(分区) `du -sh`(目录) |
| 链接 | `ln -s` 软链接最常用 |
| 批量操作 | `find -exec` 或 `xargs` |
