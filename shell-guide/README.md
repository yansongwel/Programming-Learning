## 学习建议

1. **循序渐进**：按照阶段顺序学习，每个阶段的知识都是下一阶段的基础
2. **动手实践**：每篇文章的代码都可以直接运行，一定要亲手敲一遍
3. **结合工作**：学完一个章节就尝试在日常工作中应用
4. **善用工具**：安装 ShellCheck，养成写完就检查的习惯
5. **知道边界**：Shell 脚本超过 300 行时，考虑用 Python 重写
