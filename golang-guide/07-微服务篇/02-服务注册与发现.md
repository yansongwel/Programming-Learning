# 02-服务注册与发现

## 导读

在微服务架构中，服务实例的数量和地址是动态变化的——容器可能随时被调度到不同节点，自动伸缩可能增减实例数量。硬编码服务地址显然不可行，我们需要一套机制让服务自动注册自己的位置，并让调用方自动发现目标服务。这就是**服务注册与发现**。

服务发现是 SRE/DevOps 工程师日常工作中的核心基础设施之一。无论是传统的微服务部署还是 Kubernetes 环境，理解其原理都至关重要。

**本章你将学到：**
- 服务发现的核心原理与架构模式
- etcd 作为服务注册中心的完整实践
- Consul 服务发现简介与 Go 客户端
- gRPC 客户端负载均衡
- 健康检查与故障剔除
- DNS 服务发现与对比

**前置要求：**
- 了解 gRPC 基础（上一章内容）
- 了解分布式系统基本概念

## 一、为什么需要服务发现

### 1.1 微服务通信的挑战

```
传统单体架构：
  客户端 → 固定IP:端口 → 单体应用

微服务架构：
  客户端 → ??? → 用户服务（3个实例，IP动态变化）
                → 订单服务（5个实例，随时可能扩缩容）
                → 支付服务（2个实例，可能故障重启）
```

核心问题：
1. **服务实例地址动态变化**：容器化部署下 IP 不固定
2. **实例数量动态伸缩**：根据负载自动增减
3. **故障实例需要自动剔除**：不能把请求发到故障节点
4. **需要负载均衡**：请求应该合理分配到多个实例

### 1.2 两种服务发现模式

```
客户端发现模式（Client-Side Discovery）：
  ┌──────────┐    ┌──────────────┐    ┌───────────┐
  │  客户端   │───→│  注册中心     │    │ 服务实例A  │
  │（含发现）  │    │ (etcd/Consul)│    │ 服务实例B  │
  │（含负载） │───→│              │    │ 服务实例C  │
  └──────────┘    └──────────────┘    └───────────┘
  客户端自己查询注册中心，自己选择目标实例

服务端发现模式（Server-Side Discovery）：
  ┌──────┐    ┌──────────┐    ┌──────────────┐    ┌───────────┐
  │ 客户端 │───→│ 负载均衡器 │───→│  注册中心     │    │ 服务实例A  │
  │      │    │(Nginx/LB)│    │ (etcd/Consul)│    │ 服务实例B  │
  └──────┘    └──────────┘    └──────────────┘    │ 服务实例C  │
  客户端只需知道负载均衡器地址                         └───────────┘
```

## 二、etcd 基础

### 2.1 etcd 概述

etcd 是一个高可用的分布式键值存储系统，被 Kubernetes 用作后端数据存储。它基于 Raft 共识算法保证数据一致性，非常适合作为服务注册中心。

核心特性：
- **强一致性**：基于 Raft，写入的数据保证所有节点一致
- **Watch 机制**：可以监听键的变化事件
- **Lease 租约**：键可以绑定租约，租约过期键自动删除（用于故障检测）
- **前缀查询**：支持按前缀查找所有键

### 2.2 安装与启动 etcd

```bash
# Docker 方式启动（推荐用于开发测试）
docker run -d \
  --name etcd \
  -p 2379:2379 \
  -p 2380:2380 \
  quay.io/coreos/etcd:v3.5.11 \
  /usr/local/bin/etcd \
  --name s1 \
  --data-dir /etcd-data \
  --listen-client-urls http://0.0.0.0:2379 \
  --advertise-client-urls http://0.0.0.0:2379 \
  --listen-peer-urls http://0.0.0.0:2380 \
  --initial-advertise-peer-urls http://0.0.0.0:2380 \
  --initial-cluster s1=http://0.0.0.0:2380

# 验证 etcd 是否正常
docker exec etcd etcdctl endpoint health
# 预期输出：127.0.0.1:2379 is healthy

# 基本操作
docker exec etcd etcdctl put /hello "world"
docker exec etcd etcdctl get /hello
```

### 2.3 Go etcd 客户端基础操作

```go
// etcd_basic/main.go
package main

import (
	"context"
	"fmt"
	"log"
	"time"

	clientv3 "go.etcd.io/etcd/client/v3"
)

func main() {
	// 创建 etcd 客户端
	cli, err := clientv3.New(clientv3.Config{
		Endpoints:   []string{"localhost:2379"}, // etcd 地址列表
		DialTimeout: 5 * time.Second,            // 连接超时
	})
	if err != nil {
		log.Fatalf("连接 etcd 失败: %v", err)
	}
	defer cli.Close()

	ctx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
	defer cancel()

	// ===== Put：写入键值 =====
	_, err = cli.Put(ctx, "/services/user-service/instance-1", `{"addr":"192.168.1.10:8080","weight":10}`)
	if err != nil {
		log.Fatalf("写入失败: %v", err)
	}
	fmt.Println("写入成功")

	_, err = cli.Put(ctx, "/services/user-service/instance-2", `{"addr":"192.168.1.11:8080","weight":20}`)
	if err != nil {
		log.Fatalf("写入失败: %v", err)
	}

	// ===== Get：读取单个键 =====
	resp, err := cli.Get(ctx, "/services/user-service/instance-1")
	if err != nil {
		log.Fatalf("读取失败: %v", err)
	}
	for _, kv := range resp.Kvs {
		fmt.Printf("键: %s, 值: %s, 版本: %d\n", kv.Key, kv.Value, kv.Version)
	}

	// ===== 前缀查询：获取某个前缀下的所有键 =====
	resp, err = cli.Get(ctx, "/services/user-service/", clientv3.WithPrefix())
	if err != nil {
		log.Fatalf("前缀查询失败: %v", err)
	}
	fmt.Printf("\n前缀查询结果 (共 %d 个):\n", resp.Count)
	for _, kv := range resp.Kvs {
		fmt.Printf("  %s = %s\n", kv.Key, kv.Value)
	}

	// ===== Delete：删除键 =====
	dresp, err := cli.Delete(ctx, "/services/user-service/instance-2")
	if err != nil {
		log.Fatalf("删除失败: %v", err)
	}
	fmt.Printf("\n删除了 %d 个键\n", dresp.Deleted)

	// ===== Watch：监听键变化 =====
	fmt.Println("\n开始监听 /services/user-service/ 前缀的变化...")
	go func() {
		// 模拟其他进程修改 etcd
		time.Sleep(1 * time.Second)
		cli.Put(context.Background(), "/services/user-service/instance-3",
			`{"addr":"192.168.1.12:8080"}`)
		time.Sleep(1 * time.Second)
		cli.Delete(context.Background(), "/services/user-service/instance-3")
	}()

	watchChan := cli.Watch(context.Background(), "/services/user-service/",
		clientv3.WithPrefix())

	// 设置一个超时，避免永久阻塞
	timer := time.NewTimer(5 * time.Second)
	defer timer.Stop()

watchLoop:
	for {
		select {
		case watchResp := <-watchChan:
			for _, event := range watchResp.Events {
				switch event.Type {
				case clientv3.EventTypePut:
					fmt.Printf("  [PUT] %s = %s\n", event.Kv.Key, event.Kv.Value)
				case clientv3.EventTypeDelete:
					fmt.Printf("  [DELETE] %s\n", event.Kv.Key)
				}
			}
		case <-timer.C:
			fmt.Println("监听超时退出")
			break watchLoop
		}
	}

	// ===== Lease 租约 =====
	fmt.Println("\n===== 租约演示 =====")
	leaseDemo(cli)
}

// leaseDemo 展示 etcd 租约机制
func leaseDemo(cli *clientv3.Client) {
	ctx := context.Background()

	// 创建一个 5 秒的租约
	leaseResp, err := cli.Grant(ctx, 5) // TTL = 5 秒
	if err != nil {
		log.Fatalf("创建租约失败: %v", err)
	}
	fmt.Printf("创建租约成功, ID: %x, TTL: %d秒\n", leaseResp.ID, leaseResp.TTL)

	// 将键绑定到租约（租约过期，键自动删除）
	_, err = cli.Put(ctx, "/services/temp-service/instance-1",
		`{"addr":"10.0.0.1:8080"}`,
		clientv3.WithLease(leaseResp.ID))
	if err != nil {
		log.Fatalf("带租约写入失败: %v", err)
	}
	fmt.Println("带租约写入成功")

	// 查看键存在
	resp, _ := cli.Get(ctx, "/services/temp-service/instance-1")
	fmt.Printf("租约存活时，键存在: %d 个\n", resp.Count)

	// KeepAlive：持续续约（保持服务注册有效）
	keepAliveChan, err := cli.KeepAlive(ctx, leaseResp.ID)
	if err != nil {
		log.Fatalf("续约失败: %v", err)
	}

	// 读取续约响应（必须消费，否则会阻塞）
	go func() {
		for ka := range keepAliveChan {
			if ka == nil {
				fmt.Println("续约通道关闭")
				return
			}
			// 正常情况下不需要打印，这里仅做演示
			fmt.Printf("续约成功, TTL: %d\n", ka.TTL)
		}
	}()

	// 让续约运行几秒
	time.Sleep(3 * time.Second)

	// 撤销租约（模拟服务下线）
	_, err = cli.Revoke(ctx, leaseResp.ID)
	if err != nil {
		log.Printf("撤销租约失败: %v", err)
	}
	fmt.Println("租约已撤销")

	// 验证键已被删除
	resp, _ = cli.Get(ctx, "/services/temp-service/instance-1")
	fmt.Printf("租约撤销后，键存在: %d 个\n", resp.Count)
}
```

## 三、基于 etcd 的服务注册与发现

### 3.1 服务注册器

```go
// registry/etcd_registry.go
package registry

import (
	"context"
	"encoding/json"
	"fmt"
	"log"
	"sync"
	"time"

	clientv3 "go.etcd.io/etcd/client/v3"
)

// ServiceInstance 服务实例信息
type ServiceInstance struct {
	ID       string            `json:"id"`       // 实例唯一标识
	Name     string            `json:"name"`     // 服务名
	Addr     string            `json:"addr"`     // 地址 host:port
	Weight   int               `json:"weight"`   // 权重（用于负载均衡）
	Metadata map[string]string `json:"metadata"` // 扩展元数据
}

// EtcdRegistry 基于 etcd 的服务注册器
type EtcdRegistry struct {
	client    *clientv3.Client
	leaseID   clientv3.LeaseID
	leaseTTL  int64  // 租约 TTL（秒）
	prefix    string // 键前缀，如 "/services/"

	cancelFunc context.CancelFunc
	mu         sync.Mutex
	registered bool
}

// NewEtcdRegistry 创建 etcd 注册器
func NewEtcdRegistry(endpoints []string, leaseTTL int64) (*EtcdRegistry, error) {
	client, err := clientv3.New(clientv3.Config{
		Endpoints:   endpoints,
		DialTimeout: 5 * time.Second,
	})
	if err != nil {
		return nil, fmt.Errorf("连接 etcd 失败: %w", err)
	}

	return &EtcdRegistry{
		client:   client,
		leaseTTL: leaseTTL,
		prefix:   "/services/",
	}, nil
}

// Register 注册服务实例
func (r *EtcdRegistry) Register(ctx context.Context, instance *ServiceInstance) error {
	r.mu.Lock()
	defer r.mu.Unlock()

	if r.registered {
		return fmt.Errorf("该实例已注册")
	}

	// 序列化实例信息
	value, err := json.Marshal(instance)
	if err != nil {
		return fmt.Errorf("序列化实例信息失败: %w", err)
	}

	// 构建 etcd 键：/services/{服务名}/{实例ID}
	key := fmt.Sprintf("%s%s/%s", r.prefix, instance.Name, instance.ID)

	// 创建租约
	leaseResp, err := r.client.Grant(ctx, r.leaseTTL)
	if err != nil {
		return fmt.Errorf("创建租约失败: %w", err)
	}
	r.leaseID = leaseResp.ID

	// 将服务信息写入 etcd（绑定租约）
	_, err = r.client.Put(ctx, key, string(value), clientv3.WithLease(r.leaseID))
	if err != nil {
		return fmt.Errorf("注册服务失败: %w", err)
	}

	// 启动自动续约
	keepAliveCtx, cancelFunc := context.WithCancel(context.Background())
	r.cancelFunc = cancelFunc

	keepAliveChan, err := r.client.KeepAlive(keepAliveCtx, r.leaseID)
	if err != nil {
		cancelFunc()
		return fmt.Errorf("启动续约失败: %w", err)
	}

	// 异步消费续约响应
	go func() {
		for {
			select {
			case resp, ok := <-keepAliveChan:
				if !ok {
					log.Printf("[注册中心] 续约通道关闭, 服务: %s, 实例: %s", instance.Name, instance.ID)
					return
				}
				if resp == nil {
					log.Printf("[注册中心] 续约失败, 服务: %s, 实例: %s", instance.Name, instance.ID)
					return
				}
				// 续约成功，通常不需要打印
			case <-keepAliveCtx.Done():
				return
			}
		}
	}()

	r.registered = true
	log.Printf("[注册中心] 服务注册成功: %s (实例: %s, 地址: %s)", instance.Name, instance.ID, instance.Addr)
	return nil
}

// Deregister 注销服务实例
func (r *EtcdRegistry) Deregister(ctx context.Context, instance *ServiceInstance) error {
	r.mu.Lock()
	defer r.mu.Unlock()

	if !r.registered {
		return nil
	}

	// 停止续约
	if r.cancelFunc != nil {
		r.cancelFunc()
	}

	// 撤销租约（自动删除绑定的键）
	if _, err := r.client.Revoke(ctx, r.leaseID); err != nil {
		return fmt.Errorf("撤销租约失败: %w", err)
	}

	r.registered = false
	log.Printf("[注册中心] 服务注销成功: %s (实例: %s)", instance.Name, instance.ID)
	return nil
}

// Close 关闭注册器
func (r *EtcdRegistry) Close() error {
	return r.client.Close()
}
```

### 3.2 服务发现器

```go
// registry/etcd_discovery.go
package registry

import (
	"context"
	"encoding/json"
	"fmt"
	"log"
	"sync"

	clientv3 "go.etcd.io/etcd/client/v3"
)

// ServiceDiscovery 服务发现器
type ServiceDiscovery struct {
	client    *clientv3.Client
	prefix    string
	instances map[string][]*ServiceInstance // 服务名 -> 实例列表
	mu        sync.RWMutex
}

// NewServiceDiscovery 创建服务发现器
func NewServiceDiscovery(client *clientv3.Client) *ServiceDiscovery {
	return &ServiceDiscovery{
		client:    client,
		prefix:    "/services/",
		instances: make(map[string][]*ServiceInstance),
	}
}

// Discover 发现指定服务的所有实例
func (d *ServiceDiscovery) Discover(ctx context.Context, serviceName string) ([]*ServiceInstance, error) {
	prefix := fmt.Sprintf("%s%s/", d.prefix, serviceName)

	// 从 etcd 获取前缀下的所有键
	resp, err := d.client.Get(ctx, prefix, clientv3.WithPrefix())
	if err != nil {
		return nil, fmt.Errorf("查询服务失败: %w", err)
	}

	instances := make([]*ServiceInstance, 0, resp.Count)
	for _, kv := range resp.Kvs {
		var instance ServiceInstance
		if err := json.Unmarshal(kv.Value, &instance); err != nil {
			log.Printf("反序列化实例信息失败: %v (key=%s)", err, kv.Key)
			continue
		}
		instances = append(instances, &instance)
	}

	// 缓存到本地
	d.mu.Lock()
	d.instances[serviceName] = instances
	d.mu.Unlock()

	return instances, nil
}

// Watch 监听服务实例变化
func (d *ServiceDiscovery) Watch(ctx context.Context, serviceName string, onChange func([]*ServiceInstance)) {
	prefix := fmt.Sprintf("%s%s/", d.prefix, serviceName)

	// 先获取当前所有实例
	instances, err := d.Discover(ctx, serviceName)
	if err != nil {
		log.Printf("初始发现失败: %v", err)
	} else {
		onChange(instances)
	}

	// 监听变化
	watchChan := d.client.Watch(ctx, prefix, clientv3.WithPrefix())

	go func() {
		for {
			select {
			case <-ctx.Done():
				log.Printf("停止监听服务: %s", serviceName)
				return
			case watchResp, ok := <-watchChan:
				if !ok {
					log.Printf("监听通道关闭: %s", serviceName)
					return
				}

				for _, event := range watchResp.Events {
					switch event.Type {
					case clientv3.EventTypePut:
						var instance ServiceInstance
						if err := json.Unmarshal(event.Kv.Value, &instance); err != nil {
							continue
						}
						d.addInstance(serviceName, &instance)
						log.Printf("[发现] 服务上线: %s -> %s", serviceName, instance.Addr)

					case clientv3.EventTypeDelete:
						d.removeInstanceByKey(serviceName, string(event.Kv.Key))
						log.Printf("[发现] 服务下线: %s -> key=%s", serviceName, event.Kv.Key)
					}
				}

				// 通知变化
				d.mu.RLock()
				currentInstances := d.instances[serviceName]
				d.mu.RUnlock()
				onChange(currentInstances)
			}
		}
	}()
}

// addInstance 添加或更新实例
func (d *ServiceDiscovery) addInstance(serviceName string, instance *ServiceInstance) {
	d.mu.Lock()
	defer d.mu.Unlock()

	instances := d.instances[serviceName]
	// 更新已有实例或添加新实例
	for i, inst := range instances {
		if inst.ID == instance.ID {
			instances[i] = instance
			return
		}
	}
	d.instances[serviceName] = append(instances, instance)
}

// removeInstanceByKey 根据 etcd 键删除实例
func (d *ServiceDiscovery) removeInstanceByKey(serviceName, key string) {
	d.mu.Lock()
	defer d.mu.Unlock()

	instances := d.instances[serviceName]
	for i, inst := range instances {
		expectedKey := fmt.Sprintf("%s%s/%s", d.prefix, serviceName, inst.ID)
		if expectedKey == key {
			// 从切片中删除
			d.instances[serviceName] = append(instances[:i], instances[i+1:]...)
			return
		}
	}
}

// GetInstances 获取缓存的实例列表
func (d *ServiceDiscovery) GetInstances(serviceName string) []*ServiceInstance {
	d.mu.RLock()
	defer d.mu.RUnlock()
	return d.instances[serviceName]
}
```

### 3.3 完整使用示例

```go
// main.go
package main

import (
	"context"
	"fmt"
	"log"
	"os"
	"os/signal"
	"syscall"
	"time"

	clientv3 "go.etcd.io/etcd/client/v3"
)

func main() {
	endpoints := []string{"localhost:2379"}

	// ===== 模拟服务端：注册服务 =====
	registry, err := NewEtcdRegistry(endpoints, 10) // TTL 10秒
	if err != nil {
		log.Fatal(err)
	}
	defer registry.Close()

	instance := &ServiceInstance{
		ID:     "instance-001",
		Name:   "user-service",
		Addr:   "192.168.1.10:8080",
		Weight: 10,
		Metadata: map[string]string{
			"version": "v1.2.0",
			"region":  "cn-east",
		},
	}

	ctx := context.Background()
	if err := registry.Register(ctx, instance); err != nil {
		log.Fatal(err)
	}

	// ===== 模拟客户端：发现服务 =====
	client, err := clientv3.New(clientv3.Config{
		Endpoints:   endpoints,
		DialTimeout: 5 * time.Second,
	})
	if err != nil {
		log.Fatal(err)
	}
	defer client.Close()

	discovery := NewServiceDiscovery(client)

	// 监听服务变化
	watchCtx, watchCancel := context.WithCancel(ctx)
	defer watchCancel()

	discovery.Watch(watchCtx, "user-service", func(instances []*ServiceInstance) {
		fmt.Printf("\n服务实例列表更新 (共 %d 个):\n", len(instances))
		for _, inst := range instances {
			fmt.Printf("  ID=%s, 地址=%s, 权重=%d\n", inst.ID, inst.Addr, inst.Weight)
		}
	})

	// 等待退出信号
	quit := make(chan os.Signal, 1)
	signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)

	fmt.Println("\n服务运行中，按 Ctrl+C 退出...")
	<-quit

	// 优雅注销
	fmt.Println("\n正在注销服务...")
	if err := registry.Deregister(ctx, instance); err != nil {
		log.Printf("注销失败: %v", err)
	}
}
```

## 四、Consul 服务发现

### 4.1 Consul 简介

Consul 是 HashiCorp 开发的服务网格解决方案，提供服务发现、配置管理、健康检查等功能。与 etcd 相比，Consul 内置了更丰富的服务发现功能。

```bash
# Docker 启动 Consul（开发模式）
docker run -d \
  --name consul \
  -p 8500:8500 \
  -p 8600:8600/udp \
```

### 5.1 手动实现 Round-Robin

```go
// balancer/round_robin.go
package balancer

import (
	"sync"
	"sync/atomic"
)

// RoundRobinBalancer 轮询负载均衡器
type RoundRobinBalancer struct {
	instances []string  // 服务实例地址列表
	index     uint64    // 当前索引（原子操作）
	mu        sync.RWMutex
}

// NewRoundRobinBalancer 创建轮询负载均衡器
func NewRoundRobinBalancer() *RoundRobinBalancer {
	return &RoundRobinBalancer{
		instances: make([]string, 0),
	}
}

// Update 更新实例列表
func (b *RoundRobinBalancer) Update(instances []string) {
	b.mu.Lock()
	defer b.mu.Unlock()
	b.instances = instances
}

// Pick 选择下一个实例
func (b *RoundRobinBalancer) Pick() (string, bool) {
	b.mu.RLock()
	defer b.mu.RUnlock()

	if len(b.instances) == 0 {
		return "", false
	}

	// 原子递增并取模，实现轮询
	idx := atomic.AddUint64(&b.index, 1)
	instance := b.instances[idx%uint64(len(b.instances))]
	return instance, true
}

// WeightedRoundRobinBalancer 加权轮询负载均衡器
type WeightedRoundRobinBalancer struct {
	instances []WeightedInstance
	mu        sync.RWMutex
}

// WeightedInstance 带权重的实例
type WeightedInstance struct {
	Addr            string
	Weight          int // 配置权重
	CurrentWeight   int // 当前权重（算法用）
	EffectiveWeight int // 有效权重
}

// NewWeightedRoundRobinBalancer 创建加权轮询负载均衡器
func NewWeightedRoundRobinBalancer() *WeightedRoundRobinBalancer {
	return &WeightedRoundRobinBalancer{}
}

// Update 更新实例列表
func (b *WeightedRoundRobinBalancer) Update(instances []WeightedInstance) {
	b.mu.Lock()
	defer b.mu.Unlock()
	b.instances = instances
	// 初始化有效权重
	for i := range b.instances {
		b.instances[i].EffectiveWeight = b.instances[i].Weight
	}
}

// Pick 使用 Nginx 的平滑加权轮询算法选择实例
func (b *WeightedRoundRobinBalancer) Pick() (string, bool) {
	b.mu.Lock()
	defer b.mu.Unlock()

	if len(b.instances) == 0 {
		return "", false
	}

	totalWeight := 0
	var best *WeightedInstance

	for i := range b.instances {
		inst := &b.instances[i]
		totalWeight += inst.EffectiveWeight
		inst.CurrentWeight += inst.EffectiveWeight

		if best == nil || inst.CurrentWeight > best.CurrentWeight {
			best = inst
		}
	}

	if best == nil {
		return "", false
	}

	best.CurrentWeight -= totalWeight
	return best.Addr, true
}
```

```go
// resolver/etcd_resolver.go
package resolver

import (
	"context"
	"encoding/json"
	"fmt"
	"log"
	"time"

	clientv3 "go.etcd.io/etcd/client/v3"
	"google.golang.org/grpc/resolver"
)

const etcdScheme = "etcd"

// EtcdResolverBuilder 实现 resolver.Builder 接口
// gRPC 通过 Builder 模式创建 Resolver
type EtcdResolverBuilder struct {
	etcdClient *clientv3.Client
	prefix     string
}

// NewEtcdResolverBuilder 创建 etcd resolver builder
func NewEtcdResolverBuilder(endpoints []string) (*EtcdResolverBuilder, error) {
	client, err := clientv3.New(clientv3.Config{
		Endpoints:   endpoints,
		DialTimeout: 5 * time.Second,
	})
	if err != nil {
		return nil, err
	}

	return &EtcdResolverBuilder{
		etcdClient: client,
		prefix:     "/services/",
	}, nil
}

// Build 创建 resolver（gRPC 在 Dial 时调用）
func (b *EtcdResolverBuilder) Build(
	target resolver.Target,
	cc resolver.ClientConn,
	opts resolver.BuildOptions,
) (resolver.Resolver, error) {
	// target.Endpoint() 返回服务名，如 "user-service"
	serviceName := target.Endpoint()
	prefix := fmt.Sprintf("%s%s/", b.prefix, serviceName)

	r := &etcdResolver{
		client: b.etcdClient,
		cc:     cc,
		prefix: prefix,
	}

	// 首次解析
	r.resolve()

	// 启动 Watch 监听变化
	go r.watch()

	return r, nil
}

// Scheme 返回 scheme 名称
func (b *EtcdResolverBuilder) Scheme() string {
	return etcdScheme
}

// etcdResolver 实现 resolver.Resolver 接口
type etcdResolver struct {
	client *clientv3.Client
	cc     resolver.ClientConn
	prefix string
	cancel context.CancelFunc
}

// resolve 从 etcd 获取服务地址并更新 gRPC 连接
func (r *etcdResolver) resolve() {
	resp, err := r.client.Get(context.Background(), r.prefix, clientv3.WithPrefix())
	if err != nil {
		log.Printf("etcd resolver: 查询失败: %v", err)
		return
	}

	addresses := make([]resolver.Address, 0, resp.Count)
	for _, kv := range resp.Kvs {
		var instance struct {
			Addr string `json:"addr"`
		}
		if err := json.Unmarshal(kv.Value, &instance); err != nil {
			continue
		}
		addresses = append(addresses, resolver.Address{
			Addr: instance.Addr,
		})
	}

	// 更新 gRPC 客户端的地址列表
	if err := r.cc.UpdateState(resolver.State{Addresses: addresses}); err != nil {
		log.Printf("etcd resolver: 更新地址失败: %v", err)
	}
	log.Printf("etcd resolver: 更新地址列表，共 %d 个实例", len(addresses))
}

// watch 监听 etcd 变化
func (r *etcdResolver) watch() {
	ctx, cancel := context.WithCancel(context.Background())
	r.cancel = cancel

	watchChan := r.client.Watch(ctx, r.prefix, clientv3.WithPrefix())
	for range watchChan {
		// 有变化就重新解析
		r.resolve()
	}
}

// ResolveNow gRPC 框架调用，要求立即重新解析
func (r *etcdResolver) ResolveNow(options resolver.ResolveNowOptions) {
	r.resolve()
}

// Close 关闭 resolver
func (r *etcdResolver) Close() {
	if r.cancel != nil {
		r.cancel()
	}
}
```

## 六、健康检查机制

### 6.1 主动健康检查

```go
// healthcheck/checker.go
package healthcheck

import (
	"context"
	"fmt"
	"log"
	"net"
	"sync"
	"time"
)

// HealthStatus 健康状态
type HealthStatus int

const (
	StatusHealthy   HealthStatus = iota // 健康
	StatusUnhealthy                     // 不健康
	StatusUnknown                       // 未知
)

// InstanceHealth 实例健康信息
type InstanceHealth struct {
	Addr         string
	Status       HealthStatus
	LastCheck    time.Time
	FailCount    int           // 连续失败次数
	ResponseTime time.Duration // 响应时间
}

// HealthChecker 健康检查器
type HealthChecker struct {
	instances map[string]*InstanceHealth
	mu        sync.RWMutex

	interval      time.Duration // 检查间隔
	timeout       time.Duration // 超时时间
	maxFail       int           // 最大连续失败次数
	checkType     string        // 检查类型："tcp" 或 "http"

	onStatusChange func(addr string, status HealthStatus) // 状态变化回调
}

// NewHealthChecker 创建健康检查器
func NewHealthChecker(interval, timeout time.Duration, maxFail int) *HealthChecker {
	return &HealthChecker{
		instances: make(map[string]*InstanceHealth),
		interval:  interval,
		timeout:   timeout,
		maxFail:   maxFail,
		checkType: "tcp",
	}
}

// OnStatusChange 注册状态变化回调
func (hc *HealthChecker) OnStatusChange(fn func(addr string, status HealthStatus)) {
	hc.onStatusChange = fn
}

// AddInstance 添加监控实例
func (hc *HealthChecker) AddInstance(addr string) {
	hc.mu.Lock()
	defer hc.mu.Unlock()
	hc.instances[addr] = &InstanceHealth{
		Addr:   addr,
		Status: StatusUnknown,
	}
}

// RemoveInstance 移除监控实例
func (hc *HealthChecker) RemoveInstance(addr string) {
	hc.mu.Lock()
	defer hc.mu.Unlock()
	delete(hc.instances, addr)
}

// Start 启动健康检查
func (hc *HealthChecker) Start(ctx context.Context) {
	ticker := time.NewTicker(hc.interval)
	defer ticker.Stop()

	// 立即执行一次
	hc.checkAll()

	for {
		select {
		case <-ctx.Done():
			log.Println("健康检查器停止")
			return
		case <-ticker.C:
			hc.checkAll()
		}
	}
}

// checkAll 检查所有实例
func (hc *HealthChecker) checkAll() {
	hc.mu.RLock()
	addrs := make([]string, 0, len(hc.instances))
	for addr := range hc.instances {
		addrs = append(addrs, addr)
	}
	hc.mu.RUnlock()

	for _, addr := range addrs {
		go hc.check(addr)
	}
}

// check 检查单个实例
func (hc *HealthChecker) check(addr string) {
	start := time.Now()

	// TCP 连接检查
	conn, err := net.DialTimeout("tcp", addr, hc.timeout)
	responseTime := time.Since(start)

	hc.mu.Lock()
	defer hc.mu.Unlock()

	inst, ok := hc.instances[addr]
	if !ok {
		return
	}

	inst.LastCheck = time.Now()
	inst.ResponseTime = responseTime
	oldStatus := inst.Status

	if err != nil {
		// 检查失败
		inst.FailCount++
		if inst.FailCount >= hc.maxFail {
			inst.Status = StatusUnhealthy
		}
		log.Printf("[健康检查] %s 失败 (%d/%d): %v", addr, inst.FailCount, hc.maxFail, err)
	} else {
		// 检查成功
		conn.Close()
		inst.FailCount = 0
		inst.Status = StatusHealthy
	}

	// 状态变化回调
	if oldStatus != inst.Status && hc.onStatusChange != nil {
		go hc.onStatusChange(addr, inst.Status)
	}
}

// GetHealthyInstances 获取所有健康实例
func (hc *HealthChecker) GetHealthyInstances() []string {
	hc.mu.RLock()
	defer hc.mu.RUnlock()

	healthy := make([]string, 0)
	for _, inst := range hc.instances {
		if inst.Status == StatusHealthy {
			healthy = append(healthy, inst.Addr)
		}
	}
	return healthy
}

// GetStatus 获取所有实例状态
func (hc *HealthChecker) GetStatus() map[string]*InstanceHealth {
	hc.mu.RLock()
	defer hc.mu.RUnlock()

	result := make(map[string]*InstanceHealth, len(hc.instances))
	for k, v := range hc.instances {
		copied := *v
		result[k] = &copied
	}
	return result
}

func main() {
	checker := NewHealthChecker(
		5*time.Second,  // 每 5 秒检查一次
		2*time.Second,  // 超时 2 秒
		3,              // 连续失败 3 次标记为不健康
	)

	checker.AddInstance("localhost:8080")
	checker.AddInstance("localhost:8081")
	checker.AddInstance("localhost:8082")

	checker.OnStatusChange(func(addr string, status HealthStatus) {
		statusText := "未知"
		switch status {
		case StatusHealthy:
			statusText = "健康"
		case StatusUnhealthy:
			statusText = "不健康"
		}
		fmt.Printf("状态变化: %s -> %s\n", addr, statusText)
	})

	ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
	defer cancel()

	checker.Start(ctx)
}
```

### 坑点 3：忽略优雅关闭

```go
// 错误：直接退出进程，服务注册信息残留在 etcd 中
// 虽然租约最终会过期，但在过期前客户端可能连到已下线的实例

// 正确：监听退出信号，优雅注销
quit := make(chan os.Signal, 1)
signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
<-quit

// 优雅注销
registry.Deregister(ctx, instance)
grpcServer.GracefulStop() // gRPC 优雅停止
```

### 坑点 4：gRPC round_robin 不生效

```go
// 错误：使用 NewClient 但没设置负载均衡策略
conn, _ := grpc.NewClient("etcd:///user-service",
    grpc.WithTransportCredentials(insecure.NewCredentials()),
)
// 默认使用 pick_first，所有请求打到同一个实例

// 正确：显式指定 round_robin
conn, _ := grpc.NewClient("etcd:///user-service",
    grpc.WithTransportCredentials(insecure.NewCredentials()),
    grpc.WithDefaultServiceConfig(`{"loadBalancingPolicy":"round_robin"}`),
)
```

### 坑点 5：Watch 没处理断线重连

```go
// 错误：Watch channel 关闭后不重建
watchChan := client.Watch(ctx, prefix, clientv3.WithPrefix())
for resp := range watchChan {
    // channel 关闭后循环结束，再也收不到事件了
}

// 正确：检测 channel 关闭并重建 Watch
for {
    watchChan := client.Watch(ctx, prefix, clientv3.WithPrefix())
    for resp := range watchChan {
        if resp.Err() != nil {
            log.Printf("Watch 错误: %v，准备重建...", resp.Err())
            break
        }
        // 处理事件...
    }
    // channel 关闭，延迟后重建
    log.Println("Watch channel 关闭，1 秒后重建...")
    time.Sleep(time.Second)
}
```

## 小结

本章完整介绍了微服务中服务注册与发现的核心内容：

1. **服务发现的必要性**：微服务环境下实例地址动态变化，需要自动化的注册与发现机制
2. **etcd** 是 Kubernetes 生态中最常用的注册中心，掌握 KV 操作、Watch、Lease 是核心
3. **Consul** 提供了更丰富的内置功能（健康检查、KV、多数据中心），适合非 K8s 环境
4. **客户端负载均衡** 通过 gRPC Name Resolver 机制无缝集成，支持自定义解析逻辑
5. **健康检查** 是服务发现的关键环节，确保请求不会被路由到故障实例
6. **DNS 服务发现** 是 K8s 环境下最简单的方案，但实时性和灵活性不如 etcd/Consul
