## 1. io.Reader / io.Writer —— Go I/O 的基石

Go 的 I/O 设计哲学：**一切皆流**。只要实现了 `io.Reader` 或 `io.Writer` 接口，就能无缝接入整个 I/O 工具链。

```go
// io.Reader 接口定义
type Reader interface {
    Read(p []byte) (n int, err error)
}

// io.Writer 接口定义
type Writer interface {
    Write(p []byte) (n int, err error)
}
```

### 1.1 谁实现了这两个接口？

| 类型 | Reader | Writer | 说明 |
|------|--------|--------|------|
| `*os.File` | Yes | Yes | 文件读写 |
| `*bytes.Buffer` | Yes | Yes | 内存缓冲区 |
| `*bytes.Reader` | Yes | No | 只读字节流 |
| `*strings.Reader` | Yes | No | 只读字符串流 |
| `*bufio.Reader` | Yes | No | 带缓冲的 Reader |
| `*bufio.Writer` | No | Yes | 带缓冲的 Writer |
| `net.Conn` | Yes | Yes | 网络连接 |
| `http.Request.Body` | Yes | No | HTTP 请求体 |
| `http.ResponseWriter` | No | Yes | HTTP 响应写入 |
| `*gzip.Reader` | Yes | No | gzip 解压流 |
| `*gzip.Writer` | No | Yes | gzip 压缩流 |

### 1.2 自定义 Reader 示例

```go
package main

import (
    "fmt"
    "io"
    "strings"
)

// 自定义 Reader：将所有字母转为大写
type UpperReader struct {
    src io.Reader
}

func (u *UpperReader) Read(p []byte) (int, error) {
    n, err := u.src.Read(p)
    for i := 0; i < n; i++ {
        if p[i] >= 'a' && p[i] <= 'z' {
            p[i] -= 32 // 小写转大写
        }
    }
    return n, err
}

func main() {
    // 用 strings.Reader 作为源
    src := strings.NewReader("hello, sre world!")
    upper := &UpperReader{src: src}

    // io.ReadAll 读取全部内容
    data, err := io.ReadAll(upper)
    if err != nil {
        panic(err)
    }
    fmt.Println(string(data)) // 输出: HELLO, SRE WORLD!
}
```

## 2. os 包文件操作

### 2.1 打开与创建文件

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    // ---- 只读打开 ----
    f1, err := os.Open("/etc/hostname") // 等价于 os.OpenFile(name, os.O_RDONLY, 0)
    if err != nil {
        fmt.Println("打开失败:", err)
        return
    }
    defer f1.Close()

    buf := make([]byte, 256)
    n, _ := f1.Read(buf)
    fmt.Printf("主机名: %s\n", string(buf[:n]))

    // ---- 创建文件（覆盖写） ----
    f2, err := os.Create("/tmp/demo.txt") // 等价于 OpenFile(name, O_RDWR|O_CREATE|O_TRUNC, 0666)
    if err != nil {
        fmt.Println("创建失败:", err)
        return
    }
    defer f2.Close()

    f2.WriteString("Hello from Go\n")
    fmt.Println("文件已写入 /tmp/demo.txt")

    // ---- OpenFile：精确控制标志和权限 ----
    f3, err := os.OpenFile("/tmp/app.log",
        os.O_APPEND|os.O_CREATE|os.O_WRONLY, // 追加模式，不存在则创建
        0644,                                  // 权限: rw-r--r--
    )
    if err != nil {
        fmt.Println("打开日志文件失败:", err)
        return
    }
    defer f3.Close()

    f3.WriteString("2024-01-15 10:00:00 INFO 服务启动\n")
    fmt.Println("日志已追加")
}
```

### 2.2 文件信息与权限

```go
    "fmt"

    "time"

    fmt.Printf("是否目录: %v\n", info.IsDir())

    // 修改权限
    err = os.Chmod("/tmp/demo.txt", 0600) // 仅属主可读写
    if err != nil {
        fmt.Println("修改权限失败:", err)
    }

```

## 3. bufio 缓冲读写

`bufio` 包在底层 I/O 之上加了一层用户态缓冲区，减少系统调用次数，显著提升性能。

### 3.1 bufio.Reader

```go
package main

import (
    "bufio"
    "fmt"
    "strings"
)

func main() {
    src := strings.NewReader("第一行\n第二行\n第三行\n")

    // 默认缓冲区 4096 字节
    reader := bufio.NewReader(src)

    // ReadString：读到指定分隔符为止（包含分隔符）
    for {
        line, err := reader.ReadString('\n')
        if len(line) > 0 {
            fmt.Printf("读到: %q\n", line)
        }
        if err != nil {
            break // io.EOF 或其他错误
        }
    }
}
```

### 3.2 bufio.Scanner —— 逐行读取的首选

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    // 读取 /etc/passwd 并打印每行
    file, err := os.Open("/etc/passwd")
    if err != nil {
        fmt.Println("打开失败:", err)
        return
    }
    defer file.Close()

    scanner := bufio.NewScanner(file)

    lineNum := 0
    for scanner.Scan() {
        lineNum++
        fmt.Printf("%4d | %s\n", lineNum, scanner.Text())
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("扫描出错:", err)
    }
    fmt.Printf("\n共 %d 行\n", lineNum)
}
```

### 3.3 Scanner 自定义分隔符

```go
package main

import (
    "bufio"
    "fmt"
    "strings"
)

func main() {
    // 以逗号分隔
    data := "nginx,redis,mysql,postgres"
    scanner := bufio.NewScanner(strings.NewReader(data))
    scanner.Split(bufio.ScanWords) // 内置：ScanLines, ScanWords, ScanRunes, ScanBytes

    // 自定义 Split 函数：以逗号分隔
    scanner2 := bufio.NewScanner(strings.NewReader(data))
    scanner2.Split(func(data []byte, atEOF bool) (advance int, token []byte, err error) {
        for i := 0; i < len(data); i++ {
            if data[i] == ',' {
                return i + 1, data[:i], nil
            }
        }
        if atEOF && len(data) > 0 {
            return len(data), data, nil
        }
        return 0, nil, nil
    })

    fmt.Println("逗号分隔结果:")
    for scanner2.Scan() {
        fmt.Printf("  服务: %s\n", scanner2.Text())
    }
}
```

### 3.4 bufio.Writer

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("/tmp/buffered.txt")
    if err != nil {
        panic(err)
    }
    defer file.Close()

    // 带缓冲的 Writer，减少磁盘写入次数
    writer := bufio.NewWriter(file)

    for i := 0; i < 1000; i++ {
        fmt.Fprintf(writer, "行 %d: 这是缓冲写入的数据\n", i+1)
    }

    // 重要：必须 Flush，否则缓冲区中的数据不会写入文件！
    err = writer.Flush()
    if err != nil {
        fmt.Println("刷新缓冲区失败:", err)
        return
    }

    fmt.Println("缓冲写入完成")
}
```

## 4. io 工具函数

### 4.1 io.Copy —— 流式拷贝

```go
package main

import (
    "fmt"
    "io"
    "os"
    "strings"
)

func main() {
    // 示例 1：字符串流 → 标准输出
    src := strings.NewReader("Hello, io.Copy!\n")
    n, err := io.Copy(os.Stdout, src)
    fmt.Printf("拷贝了 %d 字节, err=%v\n", n, err)

    // 示例 2：文件拷贝
    srcFile, err := os.Open("/etc/hostname")
    if err != nil {
        panic(err)
    }
    defer srcFile.Close()

    dstFile, err := os.Create("/tmp/hostname_copy")
    if err != nil {
        panic(err)
    }
    defer dstFile.Close()

    written, err := io.Copy(dstFile, srcFile)
    if err != nil {
        panic(err)
    }
    fmt.Printf("文件拷贝完成，写入 %d 字节\n", written)
}
```

### 4.2 io.ReadAll —— 一次性读取全部

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    file, err := os.Open("/etc/hostname")
    if err != nil {
        panic(err)
    }
    defer file.Close()

    // 注意：大文件慎用，会占用大量内存
    data, err := io.ReadAll(file)
    if err != nil {
        panic(err)
    }

    fmt.Printf("内容: %s", string(data))
    fmt.Printf("长度: %d 字节\n", len(data))

```

### 4.3 io.TeeReader —— 读取同时写入副本

```go
package main

import (
    "bytes"
    "fmt"
    "io"
    "strings"
)

func main() {
    // 场景：读取 HTTP 响应体的同时保存一份副本用于日志
    original := strings.NewReader("这是模拟的 HTTP 响应体内容")

    var logBuf bytes.Buffer
    tee := io.TeeReader(original, &logBuf)

    // 正常读取处理
    data, _ := io.ReadAll(tee)
    fmt.Println("处理数据:", string(data))

    // 日志中也保存了一份
    fmt.Println("日志副本:", logBuf.String())
}
```

### 4.4 io.MultiReader —— 串联多个 Reader

```go
package main

import (
    "fmt"
    "io"
    "strings"
)

func main() {
    // 场景：给 HTTP 请求体加前缀/后缀
    header := strings.NewReader("=== BEGIN ===\n")
    body := strings.NewReader("实际数据内容\n")
    footer := strings.NewReader("=== END ===\n")

    // 串联：依次读取 header → body → footer
    combined := io.MultiReader(header, body, footer)

    data, _ := io.ReadAll(combined)
    fmt.Println(string(data))
}
```

### 4.5 io.MultiWriter —— 同时写入多个目标

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    // 场景：日志同时输出到文件和标准输出
    logFile, err := os.Create("/tmp/multi.log")
    if err != nil {
        panic(err)
    }
    defer logFile.Close()

    // 写入 multi 的数据会同时发送到 logFile 和 os.Stdout
    multi := io.MultiWriter(logFile, os.Stdout)

    fmt.Fprintln(multi, "2024-01-15 10:00:00 [INFO] 服务启动")
    fmt.Fprintln(multi, "2024-01-15 10:00:01 [INFO] 监听端口 8080")
    fmt.Fprintln(multi, "2024-01-15 10:00:02 [WARN] 磁盘使用率 85%")
}
```

### 4.6 io.Pipe —— 同步内存管道

```go
package main

import (
    "fmt"
    "io"
    "time"
)

func main() {
    // 创建管道：写入端和读取端
    pr, pw := io.Pipe()

    // 生产者 goroutine
    go func() {
        defer pw.Close()
        for i := 1; i <= 5; i++ {
            msg := fmt.Sprintf("消息 %d\n", i)
            pw.Write([]byte(msg))
            time.Sleep(200 * time.Millisecond)
        }
    }()

    // 消费者：主 goroutine
    data, _ := io.ReadAll(pr)
    fmt.Println("收到所有消息:")
    fmt.Print(string(data))
}
```

## 5. 文件逐行读取与大文件处理

### 5.1 逐行读取日志文件

```go
package main

import (
    "bufio"
    "fmt"
    "os"
    "strings"
)

// 统计日志文件中各级别的出现次数
func countLogLevels(filename string) (map[string]int, error) {
    file, err := os.Open(filename)
    if err != nil {
        return nil, err
    }
    defer file.Close()

    counts := map[string]int{
        "DEBUG": 0,
        "INFO":  0,
        "WARN":  0,
        "ERROR": 0,
        "FATAL": 0,
    }

    scanner := bufio.NewScanner(file)
    for scanner.Scan() {
        line := scanner.Text()
        for level := range counts {
            if strings.Contains(line, level) {
                counts[level]++
                break
            }
        }
    }

    return counts, scanner.Err()
}

func main() {
    // 先创建测试日志
    testLog := `/tmp/test_app.log`
    content := `2024-01-15 10:00:00 [INFO] 服务启动
2024-01-15 10:00:01 [DEBUG] 加载配置文件
2024-01-15 10:00:02 [INFO] 监听端口 8080
2024-01-15 10:00:03 [WARN] 内存使用率 80%
2024-01-15 10:00:04 [ERROR] 数据库连接超时
2024-01-15 10:00:05 [INFO] 重试连接成功
2024-01-15 10:00:06 [ERROR] 磁盘空间不足
2024-01-15 10:00:07 [FATAL] 系统崩溃
`
    os.WriteFile(testLog, []byte(content), 0644)

    counts, err := countLogLevels(testLog)
    if err != nil {
        fmt.Println("错误:", err)
        return
    }

    fmt.Println("日志级别统计:")
    for level, count := range counts {
        fmt.Printf("  %-6s: %d\n", level, count)
    }
}
```

### 5.2 处理超长行（Scanner 缓冲区限制）

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("/tmp/long_lines.txt")
    if err != nil {
        // 创建一个包含超长行的测试文件
        f, _ := os.Create("/tmp/long_lines.txt")
        longLine := make([]byte, 1024*1024) // 1MB 的行
        for i := range longLine {
            longLine[i] = 'A'
        }
        f.Write(longLine)
        f.Write([]byte("\n"))
        f.Close()

        file, _ = os.Open("/tmp/long_lines.txt")
    }
    defer file.Close()

    scanner := bufio.NewScanner(file)

    // 默认 bufio.Scanner 缓冲区是 64KB（bufio.MaxScanTokenSize）
    // 如果行超过该限制，会报 bufio.Scanner: token too long
    // 解决：自定义缓冲区大小
    maxBufSize := 10 * 1024 * 1024 // 10MB
    buf := make([]byte, 0, 64*1024)
    scanner.Buffer(buf, maxBufSize)

    lineNum := 0
    for scanner.Scan() {
        lineNum++
        line := scanner.Text()
        fmt.Printf("行 %d: 长度=%d 字节\n", lineNum, len(line))
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("扫描错误:", err)
    }
}
```

### 5.3 大文件分块处理

```go
package main

import (
    "fmt"
    "io"
    "os"
)

func main() {
    // 模拟处理 GB 级大文件：分块读取，避免一次性加载到内存
    file, err := os.Open("/etc/passwd")
    if err != nil {
        panic(err)
    }
    defer file.Close()

    chunkSize := 512 // 每次读取 512 字节（实际场景可设为 4KB/64KB/1MB）
    buf := make([]byte, chunkSize)
    totalBytes := 0
    chunkNum := 0

    for {
        n, err := file.Read(buf)
        if n > 0 {
            chunkNum++
            totalBytes += n
            // 在此处理每个块，例如计算哈希、发送到远端等
            fmt.Printf("块 %d: 读取 %d 字节\n", chunkNum, n)
        }
        if err == io.EOF {
            break
        }
        if err != nil {
            fmt.Println("读取错误:", err)
            break
        }
    }

    fmt.Printf("总计: %d 个块, %d 字节\n", chunkNum, totalBytes)
}
```

## 6. 目录遍历

### 6.1 filepath.WalkDir（推荐，Go 1.16+）

```go
package main

import (
    "fmt"
    "io/fs"
    "os"
    "path/filepath"
)

func main() {
    root := "/tmp"

    // 创建测试目录结构
    os.MkdirAll("/tmp/walktest/logs", 0755)
    os.MkdirAll("/tmp/walktest/config", 0755)
    os.WriteFile("/tmp/walktest/logs/app.log", []byte("log"), 0644)
    os.WriteFile("/tmp/walktest/logs/error.log", []byte("err"), 0644)
    os.WriteFile("/tmp/walktest/config/nginx.conf", []byte("conf"), 0644)

    root = "/tmp/walktest"

    var fileCount, dirCount int

    // WalkDir 比 Walk 更高效：不需要对每个文件调用 os.Stat
    err := filepath.WalkDir(root, func(path string, d fs.DirEntry, err error) error {
        if err != nil {
            // 权限不足等错误，跳过该目录
            fmt.Printf("跳过 %s: %v\n", path, err)
            return filepath.SkipDir
        }

        // 跳过隐藏文件/目录
        if d.Name()[0] == '.' && d.Name() != "." {
            if d.IsDir() {
                return filepath.SkipDir
            }
            return nil
        }

        if d.IsDir() {
            dirCount++
            fmt.Printf("[DIR]  %s\n", path)
        } else {
            fileCount++
            info, _ := d.Info()
            fmt.Printf("[FILE] %s (%d bytes)\n", path, info.Size())
        }
        return nil
    })

    if err != nil {
        fmt.Println("遍历错误:", err)
    }
    fmt.Printf("\n统计: %d 个目录, %d 个文件\n", dirCount, fileCount)
}
```

## 7. SRE 实战场景

### 7.1 日志轮转检查：找出超过指定大小的日志文件

```go
package main

import (
    "fmt"
    "io/fs"
    "os"
    "path/filepath"
    "strings"
)

// 查找超过指定大小的日志文件
func findLargeLogFiles(root string, maxSizeBytes int64) ([]string, error) {
    var largeFiles []string

    err := filepath.WalkDir(root, func(path string, d fs.DirEntry, err error) error {
        if err != nil {
            return nil // 跳过无权限的目录
        }
        if d.IsDir() {
            return nil
        }

        // 只检查 .log 文件
        if !strings.HasSuffix(d.Name(), ".log") {
            return nil
        }

        info, err := d.Info()
        if err != nil {
            return nil
        }

        if info.Size() > maxSizeBytes {
            largeFiles = append(largeFiles, fmt.Sprintf("%s (%d MB)", path, info.Size()/1024/1024))
        }
        return nil
    })

    return largeFiles, err
}

func main() {
    // 创建测试大文件
    os.MkdirAll("/tmp/logcheck", 0755)
    bigData := make([]byte, 5*1024*1024) // 5MB
    os.WriteFile("/tmp/logcheck/big.log", bigData, 0644)
    os.WriteFile("/tmp/logcheck/small.log", []byte("small"), 0644)

    files, err := findLargeLogFiles("/tmp/logcheck", 1*1024*1024) // 超过 1MB
    if err != nil {
        fmt.Println("错误:", err)
        return
    }

    if len(files) == 0 {
        fmt.Println("没有超大日志文件")
    } else {
        fmt.Println("需要轮转的日志文件:")
        for _, f := range files {
            fmt.Println(" ", f)
        }
    }
}
```

### 7.2 配置文件差异比较

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

// 简单的逐行比较两个文件
func diffFiles(file1, file2 string) error {
    f1, err := os.Open(file1)
    if err != nil {
        return err
    }
    defer f1.Close()

    f2, err := os.Open(file2)
    if err != nil {
        return err
    }
    defer f2.Close()

    s1 := bufio.NewScanner(f1)
    s2 := bufio.NewScanner(f2)

    lineNum := 0
    hasDiff := false

    for {
        ok1 := s1.Scan()
        ok2 := s2.Scan()

        if !ok1 && !ok2 {
            break // 两个文件都读完了
        }

        lineNum++
        line1, line2 := "", ""
        if ok1 {
            line1 = s1.Text()
        }
        if ok2 {
            line2 = s2.Text()
        }

        if line1 != line2 {
            hasDiff = true
            fmt.Printf("行 %d 不同:\n", lineNum)
            fmt.Printf("  文件1: %s\n", line1)
            fmt.Printf("  文件2: %s\n", line2)
        }
    }

    if !hasDiff {
        fmt.Println("两个文件内容相同")
    }
    return nil
}

func main() {
    // 创建两个略有不同的配置文件
    os.WriteFile("/tmp/config_old.conf", []byte("listen 80\nserver_name example.com\nroot /var/www\n"), 0644)
    os.WriteFile("/tmp/config_new.conf", []byte("listen 443\nserver_name example.com\nroot /var/www/html\n"), 0644)

    fmt.Println("配置文件差异:")
    diffFiles("/tmp/config_old.conf", "/tmp/config_new.conf")
}
```

### 7.3 实时日志尾部追踪（类似 tail -f）

```go
package main

import (
    "bufio"
    "fmt"
    "io"
    "os"
    "time"
)

// tailFollow 类似 tail -f，持续读取文件新增内容
func tailFollow(filename string, done <-chan struct{}) error {
    file, err := os.Open(filename)
    if err != nil {
        return err
    }
    defer file.Close()

    // 跳到文件末尾
    _, err = file.Seek(0, io.SeekEnd)
    if err != nil {
        return err
    }

    reader := bufio.NewReader(file)

    for {
        select {
        case <-done:
            fmt.Println("\n停止追踪")
            return nil
        default:
            line, err := reader.ReadString('\n')
            if len(line) > 0 {
                fmt.Print("[新日志] ", line)
            }
            if err != nil {
                // 到达文件末尾，等一会儿再重试
                time.Sleep(500 * time.Millisecond)
                continue
            }
        }
    }
}

func main() {
    logFile := "/tmp/tail_demo.log"
    os.WriteFile(logFile, []byte(""), 0644) // 创建空文件

    done := make(chan struct{})

    // 启动 tail -f
    go tailFollow(logFile, done)

    // 模拟日志写入
    go func() {
        f, _ := os.OpenFile(logFile, os.O_APPEND|os.O_WRONLY, 0644)
        defer f.Close()
        for i := 1; i <= 5; i++ {
            time.Sleep(800 * time.Millisecond)
            line := fmt.Sprintf("2024-01-15 10:00:%02d [INFO] 事件 %d\n", i, i)
            f.WriteString(line)
        }
        time.Sleep(500 * time.Millisecond)
        close(done)
    }()

    <-done
    time.Sleep(1 * time.Second)
    fmt.Println("程序结束")
}
```

## 8. 文件监控 —— fsnotify 简介

`fsnotify` 是 Go 生态中最常用的文件系统事件监控库（非标准库，但几乎所有需要文件监控的 Go 项目都在用它）。

```go
// 需要先安装：go get github.com/fsnotify/fsnotify
//
// 以下为示例代码，展示基本用法：
package main

import (
    "fmt"
    "log"

    "github.com/fsnotify/fsnotify"
)

func main() {
    watcher, err := fsnotify.NewWatcher()
    if err != nil {
        log.Fatal(err)
    }
    defer watcher.Close()

    // 启动事件处理 goroutine
    go func() {
        for {
            select {
            case event, ok := <-watcher.Events:
                if !ok {
                    return
                }
                // 事件类型：Create, Write, Remove, Rename, Chmod
                fmt.Printf("事件: %s 操作: %s\n", event.Name, event.Op)

                // SRE 场景：监控配置文件变更并自动重载
                if event.Op&fsnotify.Write == fsnotify.Write {
                    fmt.Println("  → 文件被修改，触发重载")
                }
            case err, ok := <-watcher.Errors:
                if !ok {
                    return
                }
                fmt.Println("错误:", err)
            }
        }
    }()

    // 监控 /etc 目录
    err = watcher.Add("/tmp")
    if err != nil {
        log.Fatal(err)
    }

    fmt.Println("开始监控 /tmp 目录...")
    // 实际使用中会阻塞主 goroutine
    select {} // 阻塞
}
```

## 9. 临时文件与临时目录

```go
package main

import (

func main() {

    os.WriteFile(tmpDir+"/data.json", []byte(`{"status":"ok"}`), 0644)

    // 用完后清理
    defer os.RemoveAll(tmpDir)

    // 读回验证
    data, _ := os.ReadFile(tmpDir + "/data.json")
```

## 10. 对比 Python / Shell

### 10.1 读取文件

| 操作 | Go | Python | Shell |
|------|-----|--------|-------|
| 读取全部 | `os.ReadFile(name)` | `open(name).read()` | `cat file` |
| 逐行读取 | `bufio.NewScanner(f)` | `for line in f:` | `while read line` |
| 写入文件 | `os.WriteFile(name,data,perm)` | `open(name,'w').write(data)` | `echo data > file` |
| 追加写入 | `os.OpenFile(name,O_APPEND\|...)` | `open(name,'a')` | `echo data >> file` |
| 文件是否存在 | `os.Stat(name)` + `os.IsNotExist` | `os.path.exists(name)` | `test -f file` |
| 目录遍历 | `filepath.WalkDir` | `os.walk()` | `find /path` |
| 文件拷贝 | `io.Copy(dst, src)` | `shutil.copy()` | `cp src dst` |

### 10.2 Go vs Python 逐行读取

```go
// Go 版本
func readLines(filename string) ([]string, error) {
    file, err := os.Open(filename)
    if err != nil {
        return nil, err
    }
    defer file.Close()

    var lines []string
    scanner := bufio.NewScanner(file)
    for scanner.Scan() {
        lines = append(lines, scanner.Text())
    }
    return lines, scanner.Err()
}
```

```python

def read_lines(filename):
    with open(filename) as f:
        return f.readlines()
```

Go 的代码虽然更长，但控制更精细：你可以设置缓冲区大小、自定义分隔符、处理超长行。

## 常见坑点

### 坑 1：忘记关闭文件

```go
// 错误写法：循环中打开文件但忘记关闭 → 文件描述符泄漏
func bad() {
    for i := 0; i < 10000; i++ {
        f, _ := os.Open("/etc/hostname")
        // 忘记 f.Close()
        _ = f
    }
    // 最终可能报 "too many open files"
}

// 正确写法：使用 defer 或抽取为函数
func good() {
    for i := 0; i < 10000; i++ {
        func() {
            f, err := os.Open("/etc/hostname")
            if err != nil {
                return
            }
            defer f.Close() // 每次循环结束时关闭
            // 处理文件...
        }()
    }
}
```

### 坑 2：Scanner 默认缓冲区太小

```go
// bufio.Scanner 默认 MaxScanTokenSize = 64KB
// 如果日志文件中有超长行（如 base64 编码的数据），会报错：
//   bufio.Scanner: token too long
//
// 解决方案：
scanner := bufio.NewScanner(file)
scanner.Buffer(make([]byte, 0, 64*1024), 10*1024*1024) // 最大 10MB
```

### 坑 3：bufio.Writer 忘记 Flush

```go
// 错误：数据还在缓冲区中，文件可能是空的！
writer := bufio.NewWriter(file)
writer.WriteString("important data")
// 缺少 writer.Flush()
file.Close() // 数据丢失！

// 正确：先 Flush 再关闭
writer := bufio.NewWriter(file)
writer.WriteString("important data")
writer.Flush() // 刷新缓冲区到文件
file.Close()
```

### 坑 4：os.ReadFile / io.ReadAll 处理大文件

```go
// 以下代码对小文件没问题，但如果文件是 10GB 的日志...
data, _ := os.ReadFile("/var/log/huge.log") // 直接 OOM
// 解决：对大文件使用流式处理（Scanner 或分块 Read）
```

### 坑 5：filepath.Walk vs WalkDir

```go
// Walk（旧版）：每个文件/目录都调用 os.Lstat，性能差
filepath.Walk(root, func(path string, info os.FileInfo, err error) error { ... })

// WalkDir（Go 1.16+，推荐）：使用 DirEntry 接口，按需获取文件信息
filepath.WalkDir(root, func(path string, d fs.DirEntry, err error) error { ... })
// 性能提升可达 2-3 倍
```

## 小结

- Go 的 I/O 设计围绕 `io.Reader` / `io.Writer` 两大接口，所有 I/O 组件都可以像积木一样自由组合。
- 日常文件操作：`os.ReadFile` / `os.WriteFile` 适合小文件；大文件用 `bufio.Scanner` 或分块 `Read`。
- `io.Copy` 是零拷贝的流式传输首选；`TeeReader` / `MultiReader` / `MultiWriter` 可构建复杂的 I/O 流水线。
- 目录遍历优先使用 `filepath.WalkDir`（Go 1.16+），性能优于 `filepath.Walk`。
- SRE 场景中，注意文件描述符泄漏、大文件 OOM、缓冲区溢出等问题。
- 临时文件使用 `os.CreateTemp`，记得用完清理。
