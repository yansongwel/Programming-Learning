1. 01 基础篇 → 02 进阶篇（重点：01-接口、02-错误处理）

全面掌握：按目录顺序 01 → 12 完整学习

## 环境要求

- **Go**: 1.22+（推荐 1.24，本文档基于 Go 1.24 验证）
- **操作系统**: Linux（Ubuntu/CentOS）、macOS、Windows
- **IDE**: GoLand 或 VS Code + Go 扩展
- **版本管理**: Go Modules（默认）

## 文档约定

- `对比 Python/Shell` — 关键概念附其他语言等价写法
- `SRE 实战` — 完整可运行的运维场景代码
- `常见坑点` — 踩过的坑和 Do's / Don'ts
- `速查表` — 常用方法速查

## 为什么学 Go？

| 特性 | 说明 |
|------|------|
| **编译型语言** | 编译为单一二进制文件，无需运行时依赖，部署极其方便 |
| **并发原生** | goroutine + channel 让并发编程简单高效 |
| **云原生第一语言** | Docker、Kubernetes、Prometheus、etcd 等均用 Go 编写 |
| **性能优异** | 接近 C/C++ 的性能，远超 Python/Ruby |
| **语法简洁** | 25 个关键字，学习曲线平缓 |
| **工具链完善** | 内置格式化、测试、性能分析、文档生成 |
| **交叉编译** | 一行命令编译 Linux/macOS/Windows 多平台二进制 |
