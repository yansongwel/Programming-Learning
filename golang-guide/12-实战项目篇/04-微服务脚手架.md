# 04-微服务脚手架实战

> 一键生成生产级 Go 微服务项目模板，集成 gRPC + HTTP Gateway + GORM + Redis + Prometheus 全栈能力。

---

## 导读

在 SRE/DevOps 的日常工作中，经常需要快速搭建新的微服务。如果每次都从零开始配置 gRPC、数据库、缓存、监控、中间件，不仅耗时而且容易遗漏关键组件。本文将带你构建一个**完整的微服务脚手架**，实现：

- **gRPC 服务**：定义 `.proto` 文件，生成 Go 代码，注册服务
- **HTTP Gateway**：通过 grpc-gateway 将 gRPC 接口暴露为 RESTful API
- **GORM 数据库层**：连接管理、Repository 模式、自动 Migration
- **Redis 缓存层**：连接管理、缓存策略（Cache-Aside）
- **中间件体系**：认证、日志、链路追踪、限流、恢复
- **Prometheus 监控**：请求计数、延迟直方图、自定义业务指标
- **Viper 配置管理**：多环境配置（dev/staging/prod）
- **Docker Compose**：一键启动完整开发环境
- **Makefile 自动化**：proto 生成、构建、测试、lint 全流程

最终我们以一个**用户服务（User Service）**为示例，串联所有组件。

**前置要求**：Go 1.24+、protoc 编译器、Docker、Docker Compose。

---

## 一、项目需求分析

### 1.1 痛点

| 问题 | 影响 |
|------|------|
| 每次新建服务都要重复配置 gRPC、DB、缓存 | 浪费 1-2 天时间 |
| 各服务的目录结构不统一 | 维护成本高，新人上手慢 |
| 缺少统一的中间件链 | 日志格式不一致、监控缺失 |
| 配置管理混乱 | 多环境部署容易出错 |
| 缺少标准化的 CI/CD 脚本 | 部署流程不可靠 |

### 1.2 目标

构建一个脚手架项目模板，满足以下要求：

1. **一键初始化**：克隆模板后只需修改服务名即可开始开发
2. **生产就绪**：内置健康检查、优雅关闭、Prometheus 监控
3. **可扩展**：中间件可插拔，组件可按需启用/禁用
4. **规范统一**：目录结构、日志格式、错误处理风格一致

### 4.2 配置加载代码

```go
// internal/config/config.go
package config

import (
    "fmt"
    "os"
    "strings"
    "time"

    "github.com/spf13/viper"
)

// Config 全局配置结构体
type Config struct {
    App       AppConfig       `mapstructure:"app"`
    GRPC      GRPCConfig      `mapstructure:"grpc"`
    HTTP      HTTPConfig      `mapstructure:"http"`
    Database  DatabaseConfig  `mapstructure:"database"`
    Redis     RedisConfig     `mapstructure:"redis"`
    Metrics   MetricsConfig   `mapstructure:"metrics"`
    Log       LogConfig       `mapstructure:"log"`
    Auth      AuthConfig      `mapstructure:"auth"`
    RateLimit RateLimitConfig `mapstructure:"ratelimit"`
}

type AppConfig struct {
    Name    string `mapstructure:"name"`
    Env     string `mapstructure:"env"`
    Version string `mapstructure:"version"`
}

type GRPCConfig struct {
    Port           int `mapstructure:"port"`
    MaxRecvMsgSize int `mapstructure:"max_recv_msg_size"`
    MaxSendMsgSize int `mapstructure:"max_send_msg_size"`
}

type HTTPConfig struct {
    Port         int           `mapstructure:"port"`
    ReadTimeout  time.Duration `mapstructure:"read_timeout"`
    WriteTimeout time.Duration `mapstructure:"write_timeout"`
}

type DatabaseConfig struct {
    Driver          string        `mapstructure:"driver"`
    Host            string        `mapstructure:"host"`
    Port            int           `mapstructure:"port"`
    Username        string        `mapstructure:"username"`
    Password        string        `mapstructure:"password"`
    DBName          string        `mapstructure:"dbname"`
    MaxOpenConns    int           `mapstructure:"max_open_conns"`
    MaxIdleConns    int           `mapstructure:"max_idle_conns"`
    ConnMaxLifetime time.Duration `mapstructure:"conn_max_lifetime"`
    LogLevel        string        `mapstructure:"log_level"`
}

// DSN 生成 MySQL 数据源名称
func (d *DatabaseConfig) DSN() string {
    return fmt.Sprintf("%s:%s@tcp(%s:%d)/%s?charset=utf8mb4&parseTime=True&loc=Local",
        d.Username, d.Password, d.Host, d.Port, d.DBName)
}

type RedisConfig struct {
    Addr         string        `mapstructure:"addr"`
    Password     string        `mapstructure:"password"`
    DB           int           `mapstructure:"db"`
    PoolSize     int           `mapstructure:"pool_size"`
    MinIdleConns int           `mapstructure:"min_idle_conns"`
    DialTimeout  time.Duration `mapstructure:"dial_timeout"`
    ReadTimeout  time.Duration `mapstructure:"read_timeout"`
    WriteTimeout time.Duration `mapstructure:"write_timeout"`
}

type MetricsConfig struct {
    Enabled bool   `mapstructure:"enabled"`
    Path    string `mapstructure:"path"`
    Port    int    `mapstructure:"port"`
}

type LogConfig struct {
    Level    string `mapstructure:"level"`
    Format   string `mapstructure:"format"`
    Output   string `mapstructure:"output"`
    FilePath string `mapstructure:"file_path"`
}

type AuthConfig struct {
    JWTSecret   string        `mapstructure:"jwt_secret"`
    TokenExpire time.Duration `mapstructure:"token_expire"`
}

type RateLimitConfig struct {
    Enabled bool `mapstructure:"enabled"`
    RPS     int  `mapstructure:"rps"`
    Burst   int  `mapstructure:"burst"`
}

// Load 加载配置文件
// 优先级：环境变量 > 指定环境配置文件 > 默认值
func Load() (*Config, error) {
    // 从环境变量确定运行环境，默认 dev
    env := os.Getenv("APP_ENV")
    if env == "" {
        env = "dev"
    }

    v := viper.New()
    v.SetConfigName(fmt.Sprintf("config.%s", env))
    v.SetConfigType("yaml")
    v.AddConfigPath("./configs")
    v.AddConfigPath("/etc/user-service")

    // 支持环境变量覆盖，前缀 USER_SERVICE
    // 例如 USER_SERVICE_DATABASE_PASSWORD 会覆盖 database.password
    v.SetEnvPrefix("USER_SERVICE")
    v.SetEnvKeyReplacer(strings.NewReplacer(".", "_"))
    v.AutomaticEnv()

    // 读取配置文件
    if err := v.ReadInConfig(); err != nil {
        return nil, fmt.Errorf("读取配置文件失败: %w", err)
    }

    var cfg Config
    if err := v.Unmarshal(&cfg); err != nil {
        return nil, fmt.Errorf("解析配置失败: %w", err)
    }

    // 替换 ${ENV_VAR} 占位符
    cfg.Database.Password = expandEnv(cfg.Database.Password)
    cfg.Redis.Password = expandEnv(cfg.Redis.Password)
    cfg.Auth.JWTSecret = expandEnv(cfg.Auth.JWTSecret)

    return &cfg, nil
}

// expandEnv 展开 ${VAR} 格式的环境变量
func expandEnv(s string) string {
    if strings.HasPrefix(s, "${") && strings.HasSuffix(s, "}") {
        key := s[2 : len(s)-1]
        if val := os.Getenv(key); val != "" {
            return val
        }
    }
    return s
}
```

```protobuf
syntax = "proto3";

package user.v1;

option go_package = "github.com/yourorg/user-service/api/proto/user/v1;userv1";

import "google/api/annotations.proto";
import "google/protobuf/timestamp.proto";
import "google/protobuf/empty.proto";

// UserService 用户服务
service UserService {
  // 创建用户
  rpc CreateUser(CreateUserRequest) returns (CreateUserResponse) {
    option (google.api.http) = {
      post: "/api/v1/users"
      body: "*"
    };
  }

  // 获取用户
  rpc GetUser(GetUserRequest) returns (GetUserResponse) {
    option (google.api.http) = {
      get: "/api/v1/users/{id}"
    };
  }

  // 更新用户
  rpc UpdateUser(UpdateUserRequest) returns (UpdateUserResponse) {
    option (google.api.http) = {
      put: "/api/v1/users/{id}"
      body: "*"
    };
  }

  // 删除用户
  rpc DeleteUser(DeleteUserRequest) returns (google.protobuf.Empty) {
    option (google.api.http) = {
      delete: "/api/v1/users/{id}"
    };
  }

  // 用户列表
  rpc ListUsers(ListUsersRequest) returns (ListUsersResponse) {
    option (google.api.http) = {
      get: "/api/v1/users"
    };
  }
}

// ========== 请求 / 响应消息 ==========

message User {
  int64                     id         = 1;
  string                    username   = 2;
  string                    email      = 3;
  string                    phone      = 4;
  UserStatus                status     = 5;
  google.protobuf.Timestamp created_at = 6;
  google.protobuf.Timestamp updated_at = 7;
}

enum UserStatus {
  USER_STATUS_UNSPECIFIED = 0;
  USER_STATUS_ACTIVE      = 1;
  USER_STATUS_INACTIVE    = 2;
  USER_STATUS_BANNED      = 3;
}

message CreateUserRequest {
  string username = 1;
  string email    = 2;
  string phone    = 3;
  string password = 4;
}

message CreateUserResponse {
  User user = 1;
}

message GetUserRequest {
  int64 id = 1;
}

message GetUserResponse {
  User user = 1;
}

message UpdateUserRequest {
  int64  id       = 1;

  string email    = 3;
```

### 5.2 Proto 代码生成脚本

```bash
#!/usr/bin/env bash
# scripts/proto-gen.sh
# 生成 gRPC + grpc-gateway 代码

set -euo pipefail

# 项目根目录
PROJECT_ROOT="$(cd "$(dirname "$0")/.." && pwd)"
PROTO_DIR="${PROJECT_ROOT}/api/proto"
OUT_DIR="${PROJECT_ROOT}/api/proto"

# 安装依赖工具（首次运行需要）
install_tools() {
    echo ">>> 安装 protoc 插件..."
    go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
    go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
    go install github.com/grpc-ecosystem/grpc-gateway/v2/protoc-gen-grpc-gateway@latest
    go install github.com/grpc-ecosystem/grpc-gateway/v2/protoc-gen-openapiv2@latest
    echo ">>> 安装完成"
}

# 生成代码
generate() {
    echo ">>> 生成 Proto 代码..."

    # 查找所有 .proto 文件
    find "${PROTO_DIR}" -name "*.proto" | while read -r proto_file; do
        echo "  处理: ${proto_file}"
        protoc \
            --proto_path="${PROTO_DIR}" \
            --proto_path="${PROJECT_ROOT}/third_party" \
            --go_out="${OUT_DIR}" \
            --go_opt=paths=source_relative \
            --go-grpc_out="${OUT_DIR}" \
            --go-grpc_opt=paths=source_relative \
            --grpc-gateway_out="${OUT_DIR}" \
            --grpc-gateway_opt=paths=source_relative \
            "${proto_file}"
    done

    echo ">>> 代码生成完成"
}

case "${1:-generate}" in
    install)
        install_tools
        ;;
    generate)
        generate
        ;;
    all)
        install_tools
        generate
        ;;
    *)
        echo "用法: $0 {install|generate|all}"
        exit 1
        ;;
esac
```

### 6.1 MySQL 连接管理

```go
// pkg/database/mysql.go
package database

import (
    "fmt"
    "time"

    "go.uber.org/zap"
    "gorm.io/driver/mysql"
    "gorm.io/gorm"
    "gorm.io/gorm/logger"

    "github.com/yourorg/user-service/internal/config"
)

// NewMySQL 初始化 MySQL 连接
func NewMySQL(cfg *config.DatabaseConfig, zapLogger *zap.Logger) (*gorm.DB, error) {
    // 配置 GORM 日志级别
    var logLevel logger.LogLevel
    switch cfg.LogLevel {
    case "silent":
        logLevel = logger.Silent
    case "error":
        logLevel = logger.Error
    case "warn":
        logLevel = logger.Warn
    default:
        logLevel = logger.Info
    }

    db, err := gorm.Open(mysql.Open(cfg.DSN()), &gorm.Config{
        Logger:                 logger.Default.LogMode(logLevel),
        SkipDefaultTransaction: true,  // 非事务操作跳过默认事务，提升性能
        PrepareStmt:            true,  // 缓存预编译语句
    })
    if err != nil {
        return nil, fmt.Errorf("连接 MySQL 失败: %w", err)
    }

    // 获取底层 sql.DB 以设置连接池
    sqlDB, err := db.DB()
    if err != nil {
        return nil, fmt.Errorf("获取底层 DB 失败: %w", err)
    }

    sqlDB.SetMaxOpenConns(cfg.MaxOpenConns)
    sqlDB.SetMaxIdleConns(cfg.MaxIdleConns)
    sqlDB.SetConnMaxLifetime(cfg.ConnMaxLifetime)

    // 测试连接
    if err := sqlDB.Ping(); err != nil {
        return nil, fmt.Errorf("Ping MySQL 失败: %w", err)
    }

    zapLogger.Info("MySQL 连接成功",
        zap.String("host", cfg.Host),
        zap.Int("port", cfg.Port),
        zap.String("database", cfg.DBName),
    )

    return db, nil
}

// Close 关闭数据库连接
func Close(db *gorm.DB) error {
    sqlDB, err := db.DB()
    if err != nil {
        return err
    }
    return sqlDB.Close()
}

// AutoMigrate 自动迁移数据库表结构
func AutoMigrate(db *gorm.DB, models ...interface{}) error {
    return db.AutoMigrate(models...)
}
```

### 6.3 Repository 层

```go
// internal/repository/user_repository.go
package repository

import (
    "context"
    "fmt"

    "gorm.io/gorm"

    "github.com/yourorg/user-service/internal/model"
)

// UserRepository 用户数据访问接口
type UserRepository interface {
    Create(ctx context.Context, user *model.User) error
    GetByID(ctx context.Context, id int64) (*model.User, error)
    GetByUsername(ctx context.Context, username string) (*model.User, error)
    Update(ctx context.Context, user *model.User) error
    Delete(ctx context.Context, id int64) error
    List(ctx context.Context, page, pageSize int, keyword string) ([]*model.User, int64, error)
}

// userRepository 用户数据访问实现
type userRepository struct {
    db *gorm.DB
}

// NewUserRepository 创建用户 Repository
func NewUserRepository(db *gorm.DB) UserRepository {
    return &userRepository{db: db}
}

// Create 创建用户
func (r *userRepository) Create(ctx context.Context, user *model.User) error {
    result := r.db.WithContext(ctx).Create(user)
    if result.Error != nil {
        return fmt.Errorf("创建用户失败: %w", result.Error)
    }
    return nil
}

// GetByID 根据 ID 查询用户
func (r *userRepository) GetByID(ctx context.Context, id int64) (*model.User, error) {
    var user model.User
    result := r.db.WithContext(ctx).First(&user, id)
    if result.Error != nil {
        if result.Error == gorm.ErrRecordNotFound {
            return nil, fmt.Errorf("用户不存在: id=%d", id)
        }
        return nil, fmt.Errorf("查询用户失败: %w", result.Error)
    }
    return &user, nil
}

// GetByUsername 根据用户名查询
func (r *userRepository) GetByUsername(ctx context.Context, username string) (*model.User, error) {
    var user model.User
    result := r.db.WithContext(ctx).Where("username = ?", username).First(&user)
    if result.Error != nil {
        if result.Error == gorm.ErrRecordNotFound {
            return nil, nil // 用户名不存在，返回 nil 而非 error
        }
        return nil, fmt.Errorf("查询用户失败: %w", result.Error)
    }
    return &user, nil
}

// Update 更新用户信息
func (r *userRepository) Update(ctx context.Context, user *model.User) error {
    result := r.db.WithContext(ctx).Save(user)
    if result.Error != nil {
        return fmt.Errorf("更新用户失败: %w", result.Error)
    }
    return nil
}

// Delete 删除用户（软删除）
func (r *userRepository) Delete(ctx context.Context, id int64) error {
    result := r.db.WithContext(ctx).Delete(&model.User{}, id)
    if result.Error != nil {
        return fmt.Errorf("删除用户失败: %w", result.Error)
    }
    if result.RowsAffected == 0 {
        return fmt.Errorf("用户不存在: id=%d", id)
    }
    return nil
}

// List 分页查询用户列表
func (r *userRepository) List(ctx context.Context, page, pageSize int, keyword string) ([]*model.User, int64, error) {
    var users []*model.User
    var total int64

    query := r.db.WithContext(ctx).Model(&model.User{})

    // 关键词搜索
    if keyword != "" {
        like := "%" + keyword + "%"
        query = query.Where("username LIKE ? OR email LIKE ?", like, like)
    }

    // 先查总数
    if err := query.Count(&total).Error; err != nil {
        return nil, 0, fmt.Errorf("统计用户数失败: %w", err)
    }

    // 分页查询
    offset := (page - 1) * pageSize
    if err := query.Offset(offset).Limit(pageSize).
        Order("id DESC").
        Find(&users).Error; err != nil {
        return nil, 0, fmt.Errorf("查询用户列表失败: %w", err)
    }

    return users, total, nil
```

```go
const (
    // 缓存键前缀
    userCachePrefix = "user:info:"
    // 默认缓存过期时间
    defaultCacheTTL = 30 * time.Minute
    // 空值缓存时间（防止缓存穿透）
    nullCacheTTL = 5 * time.Minute
)

// UserCache 用户缓存接口
type UserCache interface {
    Get(ctx context.Context, id int64) (*model.User, error)
    Set(ctx context.Context, user *model.User) error
    Delete(ctx context.Context, id int64) error
    SetNull(ctx context.Context, id int64) error  // 缓存空值，防穿透
}

type userCache struct {
    client *redis.Client
    logger *zap.Logger
}

// NewUserCache 创建用户缓存实例
func NewUserCache(client *redis.Client, logger *zap.Logger) UserCache {
    return &userCache{
        client: client,
        logger: logger,
    }
}

func (c *userCache) key(id int64) string {
    return fmt.Sprintf("%s%d", userCachePrefix, id)
}

// Get 从缓存获取用户
// 返回值说明：
//   - (*model.User, nil): 缓存命中，返回用户
//   - (nil, nil): 缓存未命中（key 不存在）
//   - (nil, ErrNullCache): 命中空值缓存（已知不存在的用户）
//   - (nil, error): 发生其他错误
func (c *userCache) Get(ctx context.Context, id int64) (*model.User, error) {
    data, err := c.client.Get(ctx, c.key(id)).Bytes()
    if err != nil {
        if err == redis.Nil {
            return nil, nil // 缓存未命中
        }
        c.logger.Warn("Redis GET 失败", zap.Int64("user_id", id), zap.Error(err))
        return nil, err
    }

    // 检查是否是空值缓存
    if string(data) == "null" {
        return nil, fmt.Errorf("null cache hit for user_id=%d", id)
    }

    var user model.User
    if err := json.Unmarshal(data, &user); err != nil {
        c.logger.Error("反序列化用户缓存失败", zap.Error(err))
        // 缓存数据损坏，删除后返回未命中
        _ = c.Delete(ctx, id)
        return nil, nil
    }

    return &user, nil
}

// Set 设置用户缓存
func (c *userCache) Set(ctx context.Context, user *model.User) error {
    data, err := json.Marshal(user)
    if err != nil {
        return fmt.Errorf("序列化用户失败: %w", err)
    }

    if err := c.client.Set(ctx, c.key(user.ID), data, defaultCacheTTL).Err(); err != nil {
        c.logger.Warn("Redis SET 失败", zap.Int64("user_id", user.ID), zap.Error(err))
        return err
    }

    return nil
}

// Delete 删除用户缓存
func (c *userCache) Delete(ctx context.Context, id int64) error {
    if err := c.client.Del(ctx, c.key(id)).Err(); err != nil {
        c.logger.Warn("Redis DEL 失败", zap.Int64("user_id", id), zap.Error(err))
        return err
    }
    return nil
}

// SetNull 缓存空值（防止缓存穿透）
func (c *userCache) SetNull(ctx context.Context, id int64) error {
    return c.client.Set(ctx, c.key(id), "null", nullCacheTTL).Err()
}
```

```go
// UnaryLogging gRPC 一元调用日志中间件
func UnaryLogging(logger *zap.Logger) grpc.UnaryServerInterceptor {
    return func(
        ctx context.Context,
        req interface{},
        info *grpc.UnaryServerInfo,
        handler grpc.UnaryHandler,
    ) (interface{}, error) {
        start := time.Now()

        // 调用下游处理器
        resp, err := handler(ctx, req)

        // 提取 gRPC 状态码
        st, _ := status.FromError(err)
        duration := time.Since(start)

        // 根据状态码选择日志级别
        fields := []zap.Field{
            zap.String("method", info.FullMethod),
            zap.String("code", st.Code().String()),
            zap.Duration("duration", duration),
        }

        if err != nil {
            logger.Error("gRPC 请求失败", append(fields, zap.Error(err))...)
        } else if duration > 500*time.Millisecond {
            logger.Warn("gRPC 请求慢", fields...)
        } else {
            logger.Info("gRPC 请求完成", fields...)
        }

        return resp, err
    }
}
```

```go
import (

    "/user.v1.UserService/CreateUser": true, // 注册不需要认证
}

// UnaryAuth gRPC 一元调用认证中间件
func UnaryAuth(jwtSecret string) grpc.UnaryServerInterceptor {
    return func(
        ctx context.Context,
        req interface{},
        info *grpc.UnaryServerInfo,
        handler grpc.UnaryHandler,
    ) (interface{}, error) {
        // 检查是否在白名单中
        if publicMethods[info.FullMethod] {
            return handler(ctx, req)
        }

        // 从 metadata 提取 token
        md, ok := metadata.FromIncomingContext(ctx)
        if !ok {
            return nil, status.Error(codes.Unauthenticated, "缺少 metadata")
        }

        authHeader := md.Get("authorization")
        if len(authHeader) == 0 {
            return nil, status.Error(codes.Unauthenticated, "缺少 authorization")
        }

        // 解析 Bearer Token
        token := strings.TrimPrefix(authHeader[0], "Bearer ")
        if token == authHeader[0] {
            return nil, status.Error(codes.Unauthenticated, "无效的 token 格式")
        }

        // 验证 JWT Token（简化实现，生产中应使用 jwt-go 等库）
        userID, err := validateToken(token, jwtSecret)
        if err != nil {
            return nil, status.Error(codes.Unauthenticated, "token 验证失败")
        }

        // 将用户 ID 注入 context
        ctx = context.WithValue(ctx, contextKeyUserID, userID)
        return handler(ctx, req)
    }
}
```

```go
// internal/middleware/tracing.go
package middleware

import (
    "context"

    "google.golang.org/grpc"
    "google.golang.org/grpc/metadata"
    "go.uber.org/zap"

    "crypto/rand"
    "encoding/hex"
)

// UnaryTracing gRPC 一元调用链路追踪中间件
// 为每个请求生成或传播 trace_id，方便跨服务日志关联
func UnaryTracing(logger *zap.Logger) grpc.UnaryServerInterceptor {
    return func(
        ctx context.Context,
        req interface{},
        info *grpc.UnaryServerInfo,
        handler grpc.UnaryHandler,
    ) (interface{}, error) {
        var traceID string

        // 从上游 metadata 提取 trace_id
        if md, ok := metadata.FromIncomingContext(ctx); ok {
            if values := md.Get("x-trace-id"); len(values) > 0 {
                traceID = values[0]
            }
        }

        // 如果没有则生成新的
        if traceID == "" {
            traceID = generateTraceID()
        }

        // 将 trace_id 注入 context
        ctx = context.WithValue(ctx, traceIDKey, traceID)

        // 添加到出站 metadata（传播给下游服务）
        ctx = metadata.AppendToOutgoingContext(ctx, "x-trace-id", traceID)

        // 注入日志字段
        reqLogger := logger.With(
            zap.String("trace_id", traceID),
            zap.String("method", info.FullMethod),
        )
        ctx = context.WithValue(ctx, loggerKey, reqLogger)

        return handler(ctx, req)
    }
}

type ctxKey string

const (
    traceIDKey ctxKey = "trace_id"
    loggerKey  ctxKey = "logger"
)

// GetTraceID 从 context 中提取 trace_id
func GetTraceID(ctx context.Context) string {
    if v, ok := ctx.Value(traceIDKey).(string); ok {
        return v
    }
    return ""
}

// GetLogger 从 context 中提取带 trace_id 的 logger
func GetLogger(ctx context.Context) *zap.Logger {
    if v, ok := ctx.Value(loggerKey).(*zap.Logger); ok {
        return v
    }
    return zap.L() // 回退到全局 logger
}

// generateTraceID 生成 16 字节的 trace ID
func generateTraceID() string {
    b := make([]byte, 16)
    _, _ = rand.Read(b)
    return hex.EncodeToString(b)
}
```

```go
package service

import (
    "context"

    userv1 "github.com/yourorg/user-service/api/proto/user/v1"
    "github.com/yourorg/user-service/internal/cache"
    "github.com/yourorg/user-service/internal/model"

    "github.com/yourorg/user-service/pkg/metrics"

// UserServiceServer 实现 gRPC UserService 接口
type UserServiceServer struct {
    userv1.UnimplementedUserServiceServer
    repo   repository.UserRepository
    cache  cache.UserCache
    logger *zap.Logger
}

// NewUserServiceServer 创建 UserService 实例
func NewUserServiceServer(
    repo repository.UserRepository,

    logger *zap.Logger,

    return &UserServiceServer{
        repo:   repo,
        cache:  cache,
        logger: logger,

// CreateUser 创建用户
func (s *UserServiceServer) CreateUser(
    ctx context.Context,
    req *userv1.CreateUserRequest,
) (*userv1.CreateUserResponse, error) {
    // 参数校验
    if req.Username == "" {
        return nil, status.Error(codes.InvalidArgument, "用户名不能为空")
    }
    if req.Email == "" {
        return nil, status.Error(codes.InvalidArgument, "邮箱不能为空")
    }
    if req.Password == "" || len(req.Password) < 6 {
        return nil, status.Error(codes.InvalidArgument, "密码长度至少 6 位")
    }

    // 检查用户名是否已存在
    existing, err := s.repo.GetByUsername(ctx, req.Username)
    if err != nil {
        return nil, status.Error(codes.Internal, "检查用户名失败")
    }
    if existing != nil {
        return nil, status.Error(codes.AlreadyExists, "用户名已存在")
    }

    // 密码哈希
    hashedPwd, err := bcrypt.GenerateFromPassword([]byte(req.Password), bcrypt.DefaultCost)
    if err != nil {
        return nil, status.Error(codes.Internal, "密码处理失败")
    }

    // 创建用户
    user := &model.User{
        Username: req.Username,
        Email:    req.Email,
        Phone:    req.Phone,
        Password: string(hashedPwd),
        Status:   model.UserStatusActive,
    }

    if err := s.repo.Create(ctx, user); err != nil {
        s.logger.Error("创建用户失败", zap.Error(err))
        return nil, status.Error(codes.Internal, "创建用户失败")
    }

    // 记录业务指标
    metrics.UserRegistrations.Inc()

    s.logger.Info("用户创建成功",
        zap.Int64("user_id", user.ID),
        zap.String("username", user.Username),
    )

    return &userv1.CreateUserResponse{
        User: modelToProto(user),
    }, nil
}

// GetUser 获取用户
func (s *UserServiceServer) GetUser(
    ctx context.Context,
    req *userv1.GetUserRequest,
) (*userv1.GetUserResponse, error) {
    if req.Id <= 0 {
        return nil, status.Error(codes.InvalidArgument, "用户 ID 无效")
    }

    // 1. 先查缓存
    cachedUser, err := s.cache.Get(ctx, req.Id)
    if err != nil {
        // 命中空值缓存，说明用户不存在
        metrics.CacheHits.WithLabelValues("null_hit").Inc()
        return nil, status.Error(codes.NotFound, "用户不存在")
    }
    if cachedUser != nil {
        // 缓存命中
        metrics.CacheHits.WithLabelValues("hit").Inc()
        return &userv1.GetUserResponse{User: modelToProto(cachedUser)}, nil
    }

    // 2. 缓存未命中，查数据库
    metrics.CacheHits.WithLabelValues("miss").Inc()
    user, err := s.repo.GetByID(ctx, req.Id)
    if err != nil {
        // 用户不存在，缓存空值防穿透
        _ = s.cache.SetNull(ctx, req.Id)
        return nil, status.Error(codes.NotFound, err.Error())
    }

    // 3. 写入缓存
    _ = s.cache.Set(ctx, user)

    return &userv1.GetUserResponse{
        User: modelToProto(user),
    }, nil
}

// UpdateUser 更新用户
func (s *UserServiceServer) UpdateUser(
    ctx context.Context,
    req *userv1.UpdateUserRequest,
) (*userv1.UpdateUserResponse, error) {
    if req.Id <= 0 {
        return nil, status.Error(codes.InvalidArgument, "用户 ID 无效")
    }

    // 查询现有用户
    user, err := s.repo.GetByID(ctx, req.Id)
    if err != nil {
        return nil, status.Error(codes.NotFound, err.Error())
    }

    // 更新字段（只更新非空字段）
    if req.Username != "" {
        user.Username = req.Username
    }
    if req.Email != "" {
        user.Email = req.Email
    }
    if req.Phone != "" {
        user.Phone = req.Phone
    }

    if err := s.repo.Update(ctx, user); err != nil {
        return nil, status.Error(codes.Internal, "更新用户失败")
    }

    // 删除缓存（Cache-Aside：写操作时先删缓存）
    _ = s.cache.Delete(ctx, req.Id)

    return &userv1.UpdateUserResponse{
        User: modelToProto(user),
    }, nil
}

// DeleteUser 删除用户
func (s *UserServiceServer) DeleteUser(
    ctx context.Context,
    req *userv1.DeleteUserRequest,
) (*emptypb.Empty, error) {
    // 注意：实际代码中需要导入 "google.golang.org/protobuf/types/known/emptypb"
    // 这里简化为直接使用
    if req.Id <= 0 {

    }

    if err := s.repo.Delete(ctx, req.Id); err != nil {
        return nil, status.Error(codes.Internal, err.Error())
    }

    // 删除缓存
    _ = s.cache.Delete(ctx, req.Id)

    s.logger.Info("用户删除成功", zap.Int64("user_id", req.Id))

    return &emptypb.Empty{}, nil
}

// ListUsers 用户列表
func (s *UserServiceServer) ListUsers(
    ctx context.Context,
    req *userv1.ListUsersRequest,
) (*userv1.ListUsersResponse, error) {
    page := int(req.Page)
    pageSize := int(req.PageSize)

    // 默认值
    if page <= 0 {
        page = 1
    }
    if pageSize <= 0 {
        pageSize = 20
    }
    if pageSize > 100 {
        pageSize = 100 // 限制最大页大小

    }
```

```go
import (

    "go.uber.org/zap"
    "google.golang.org/grpc"
    "google.golang.org/grpc/health"
    "google.golang.org/grpc/health/grpc_health_v1"

    userv1 "github.com/yourorg/user-service/api/proto/user/v1"
    "github.com/yourorg/user-service/internal/config"
    "github.com/yourorg/user-service/internal/middleware"
    "github.com/yourorg/user-service/internal/service"
)

// GRPCServer 封装 gRPC 服务器
type GRPCServer struct {
    server *grpc.Server
    port   int
    logger *zap.Logger
}

// NewGRPCServer 创建 gRPC 服务器
func NewGRPCServer(
    cfg *config.Config,
    userSvc *service.UserServiceServer,
    logger *zap.Logger,
) *GRPCServer {
    // 构建中间件链（注意顺序：最外层最先执行）
    opts := []grpc.ServerOption{
        grpc.MaxRecvMsgSize(cfg.GRPC.MaxRecvMsgSize),
        grpc.MaxSendMsgSize(cfg.GRPC.MaxSendMsgSize),
        grpc.ChainUnaryInterceptor(
            middleware.UnaryRecovery(logger),          // 1. 最外层：panic 恢复
            middleware.UnaryTracing(logger),            // 2. 链路追踪
            middleware.UnaryMetrics(),                  // 3. 监控指标
            middleware.UnaryLogging(logger),            // 4. 请求日志
            middleware.UnaryRateLimit(                  // 5. 限流
                cfg.RateLimit.RPS,
                cfg.RateLimit.Burst,
            ),
            middleware.UnaryAuth(cfg.Auth.JWTSecret),   // 6. 认证
        ),
    }

    server := grpc.NewServer(opts...)

    // 注册用户服务
    userv1.RegisterUserServiceServer(server, userSvc)

    // 注册健康检查服务
    healthSvc := health.NewServer()
    grpc_health_v1.RegisterHealthServer(server, healthSvc)
    healthSvc.SetServingStatus("user.v1.UserService", grpc_health_v1.HealthCheckResponse_SERVING)

    // 开发环境启用反射（便于 grpcurl 调试）
    if cfg.App.Env == "dev" {
        reflection.Register(server)
    }

    return &GRPCServer{
        server: server,
        port:   cfg.GRPC.Port,
        logger: logger,
    }
}

// Start 启动 gRPC 服务器
func (s *GRPCServer) Start() error {
    lis, err := net.Listen("tcp", fmt.Sprintf(":%d", s.port))
    if err != nil {
        return fmt.Errorf("监听端口 %d 失败: %w", s.port, err)
    }

    s.logger.Info("gRPC 服务器启动", zap.Int("port", s.port))
    return s.server.Serve(lis)
}

// Stop 优雅关闭 gRPC 服务器
func (s *GRPCServer) Stop() {
    s.logger.Info("gRPC 服务器正在关闭...")
    s.server.GracefulStop()
}
```

```go
// internal/server/http.go
package server

import (
    "context"
    "fmt"
    "net/http"
    "time"

    "github.com/grpc-ecosystem/grpc-gateway/v2/runtime"
    "github.com/prometheus/client_golang/prometheus/promhttp"
    "go.uber.org/zap"
    "google.golang.org/grpc"
    "google.golang.org/grpc/credentials/insecure"

    userv1 "github.com/yourorg/user-service/api/proto/user/v1"
    "github.com/yourorg/user-service/internal/config"
)

// HTTPServer 封装 HTTP Gateway 服务器
type HTTPServer struct {
    server *http.Server
    logger *zap.Logger
}

// NewHTTPServer 创建 HTTP Gateway 服务器
func NewHTTPServer(cfg *config.Config, logger *zap.Logger) (*HTTPServer, error) {
    ctx := context.Background()

    // 创建 gRPC Gateway Mux
    gwMux := runtime.NewServeMux(
        // 自定义错误处理
        runtime.WithErrorHandler(customHTTPError),
    )

    // 连接到本地 gRPC 服务
    grpcAddr := fmt.Sprintf("localhost:%d", cfg.GRPC.Port)
    opts := []grpc.DialOption{
        grpc.WithTransportCredentials(insecure.NewCredentials()),
    }

    // 注册 gRPC Gateway 处理器
    if err := userv1.RegisterUserServiceHandlerFromEndpoint(ctx, gwMux, grpcAddr, opts); err != nil {
        return nil, fmt.Errorf("注册 HTTP Gateway 失败: %w", err)
    }

    // 创建主路由
    mux := http.NewServeMux()

    // gRPC Gateway 路由
    mux.Handle("/api/", gwMux)

    // Prometheus 指标端点
    if cfg.Metrics.Enabled {
        mux.Handle(cfg.Metrics.Path, promhttp.Handler())
    }

    // 健康检查端点
    mux.HandleFunc("/healthz", func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Content-Type", "application/json")
        w.WriteHeader(http.StatusOK)
        _, _ = w.Write([]byte(`{"status":"ok"}`))
    })

    // 就绪检查端点
    mux.HandleFunc("/readyz", func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Content-Type", "application/json")
        w.WriteHeader(http.StatusOK)
        _, _ = w.Write([]byte(`{"status":"ready"}`))
    })

    server := &http.Server{
        Addr:         fmt.Sprintf(":%d", cfg.HTTP.Port),
        Handler:      mux,
        ReadTimeout:  cfg.HTTP.ReadTimeout,
        WriteTimeout: cfg.HTTP.WriteTimeout,
    }

    return &HTTPServer{
        server: server,
        logger: logger,
    }, nil
}

// Start 启动 HTTP 服务器
func (s *HTTPServer) Start() error {
    s.logger.Info("HTTP Gateway 启动", zap.String("addr", s.server.Addr))
    if err := s.server.ListenAndServe(); err != nil && err != http.ErrServerClosed {
        return fmt.Errorf("HTTP 服务器启动失败: %w", err)
    }
    return nil
}

// Stop 优雅关闭 HTTP 服务器
func (s *HTTPServer) Stop() {
    ctx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
    defer cancel()

    s.logger.Info("HTTP Gateway 正在关闭...")
    if err := s.server.Shutdown(ctx); err != nil {
        s.logger.Error("HTTP 关闭失败", zap.Error(err))
    }
}

// customHTTPError 自定义 HTTP 错误响应格式
func customHTTPError(
    ctx context.Context,
    mux *runtime.ServeMux,
    marshaler runtime.Marshaler,
    w http.ResponseWriter,
    r *http.Request,
    err error,
) {
    // 使用默认处理（也可以自定义 JSON 错误格式）
    runtime.DefaultHTTPErrorHandler(ctx, mux, marshaler, w, r, err)
}
```

```go
import (

func main() {
    // ========== 1. 加载配置 ==========
    cfg, err := config.Load()
    if err != nil {
        panic("加载配置失败: " + err.Error())
    }

    // ========== 2. 初始化日志 ==========
    logger := initLogger(cfg)
    defer func() { _ = logger.Sync() }()

    logger.Info("服务启动中...",
        zap.String("name", cfg.App.Name),
        zap.String("env", cfg.App.Env),
        zap.String("version", cfg.App.Version),
    )

    // ========== 3. 初始化 MySQL ==========
    db, err := database.NewMySQL(&cfg.Database, logger)
    if err != nil {
        logger.Fatal("初始化 MySQL 失败", zap.Error(err))
    }
    defer database.Close(db)

    // 自动迁移
    if err := database.AutoMigrate(db, &model.User{}); err != nil {
        logger.Fatal("数据库迁移失败", zap.Error(err))
    }
    logger.Info("数据库迁移完成")

    // ========== 4. 初始化 Redis ==========
    redisClient, err := pkgredis.NewRedisClient(&cfg.Redis, logger)
    if err != nil {
        logger.Fatal("初始化 Redis 失败", zap.Error(err))
    }
    defer redisClient.Close()

    // ========== 5. 初始化各层组件 ==========
    userRepo := repository.NewUserRepository(db)
    userCache := cache.NewUserCache(redisClient, logger)
    userSvc := service.NewUserServiceServer(userRepo, userCache, logger)

    // ========== 6. 启动 gRPC 服务器 ==========
    grpcServer := server.NewGRPCServer(cfg, userSvc, logger)
    go func() {
        if err := grpcServer.Start(); err != nil {
            logger.Fatal("gRPC 服务器启动失败", zap.Error(err))
        }
    }()

    // ========== 7. 启动 HTTP Gateway ==========
    httpServer, err := server.NewHTTPServer(cfg, logger)
    if err != nil {
        logger.Fatal("创建 HTTP Gateway 失败", zap.Error(err))
    }
    go func() {
        if err := httpServer.Start(); err != nil {
            logger.Fatal("HTTP Gateway 启动失败", zap.Error(err))
        }
    }()

    // ========== 8. 优雅关闭 ==========
    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    sig := <-quit

    logger.Info("收到关闭信号", zap.String("signal", sig.String()))

    // 先关闭 HTTP（停止接收新请求）
    httpServer.Stop()
    // 再关闭 gRPC（等待已有请求完成）
    grpcServer.Stop()

    logger.Info("服务已安全关闭")
}

// initLogger 初始化 zap 日志
func initLogger(cfg *config.Config) *zap.Logger {
    var level zapcore.Level
    switch cfg.Log.Level {
    case "debug":
        level = zapcore.DebugLevel
    case "warn":
        level = zapcore.WarnLevel
    case "error":
        level = zapcore.ErrorLevel
    default:
        level = zapcore.InfoLevel
    }

    var zapCfg zap.Config
    if cfg.Log.Format == "json" {
        zapCfg = zap.NewProductionConfig()
    } else {
        zapCfg = zap.NewDevelopmentConfig()
    }
    zapCfg.Level = zap.NewAtomicLevelAt(level)

    // 文件输出
    if cfg.Log.Output == "file" && cfg.Log.FilePath != "" {
        zapCfg.OutputPaths = []string{cfg.Log.FilePath}
    }

    logger, err := zapCfg.Build()
    if err != nil {
        panic("初始化日志失败: " + err.Error())
    }

    // 替换全局 logger
    zap.ReplaceGlobals(logger)

    return logger
}
```

## 十六、实战场景：完整启动流程

### 场景一：本地开发

```bash
# 1. 克隆项目
git clone https://github.com/yourorg/user-service.git
cd user-service

# 2. 安装依赖
go mod tidy

# 3. 安装 proto 工具并生成代码
make proto-install
make proto

# 4. 启动基础设施（MySQL + Redis + Prometheus + Grafana）
make docker-up

# 5. 等待服务就绪后运行
make run
```

## 十七、常见坑点

### 坑点 1：gRPC Gateway 注册时机

```go
// 错误：先启动 gRPC 服务器再注册 Gateway
// 如果 gRPC 还没启动完毕，Gateway 连接会失败

// 正确做法：确保 gRPC 已经在监听后再启动 Gateway
// main.go 中的处理方式：
//   1. 先 go grpcServer.Start()  -- 异步启动
//   2. 再创建并启动 HTTP Gateway -- Gateway 内部会连接 gRPC
// 注意：如果 gRPC 启动较慢，可以加入重试或等待逻辑
```

### 坑点 2：GORM 软删除的查询陷阱

```go
// GORM 使用 gorm.DeletedAt 字段时，所有查询自动加上 WHERE deleted_at IS NULL
// 如果你需要查询被软删除的记录：

// 错误：直接查询，永远看不到被删除的记录
db.Find(&users)

// 正确：使用 Unscoped() 跳过软删除过滤
db.Unscoped().Find(&users)

// 物理删除（彻底删除数据库记录）
db.Unscoped().Delete(&user, id)
```

### 坑点 3：Redis 缓存与数据库一致性

```go
// 错误的缓存更新顺序（先更新缓存，再更新数据库）
// 如果数据库更新失败，缓存中是脏数据

// 正确的 Cache-Aside 模式：
// 读取：缓存命中 -> 返回；缓存未命中 -> 查数据库 -> 写缓存 -> 返回
// 写入：更新数据库 -> 删除缓存（而非更新缓存）

// 为什么是"删除"而非"更新"？
// 因为删除操作是幂等的，即使删除失败也只是导致下次多查一次数据库
// 而更新缓存如果并发执行可能导致旧值覆盖新值
```

### 坑点 4：gRPC 状态码映射

```go
// gRPC 状态码与 HTTP 状态码的映射并非一一对应，容易混淆：
//
// gRPC Code             HTTP Status    使用场景
// -------------------------------------------------------------------
// codes.OK              200            成功
// codes.InvalidArgument 400            参数校验失败
// codes.NotFound        404            资源不存在
// codes.AlreadyExists   409            资源已存在（如用户名重复）
// codes.Unauthenticated 401            未认证（缺少/无效 token）
// codes.PermissionDenied 403           无权限
// codes.ResourceExhausted 429          限流/配额用尽
// codes.Internal        500            服务内部错误
// codes.Unavailable     503            服务不可用
//
// 常见错误：用 codes.Internal 返回所有错误
// 正确做法：根据错误原因选择合适的状态码
```

### 坑点 5：连接池大小不当

```go
// MySQL 连接池配置不当会导致以下问题：
//
// 1. MaxOpenConns 太小：高并发时请求排队，延迟飙升
// 2. MaxOpenConns 太大：MySQL 连接数耗尽（默认 max_connections=151）
// 3. MaxIdleConns > MaxOpenConns：空闲连接永远不会超过 MaxOpenConns，设置无效
// 4. ConnMaxLifetime 太长：MySQL 端超时断开后，Go 端还在复用已断开的连接

// 推荐配置（根据实际 QPS 调整）：
// 开发环境：MaxOpen=20, MaxIdle=10, Lifetime=5min
// 生产环境：MaxOpen=100, MaxIdle=50, Lifetime=5min
// 规则：MaxIdle = MaxOpen / 2，Lifetime < MySQL wait_timeout（默认 8h）
```

### 坑点 6：Protobuf 零值语义

```go
// Protobuf 中的零值不会被序列化，这会导致意外行为：
//
// 例：用户 status = 0 时，proto 序列化后该字段"消失"
// 解决方案：枚举第一个值设为 UNSPECIFIED，业务值从 1 开始
//
// enum UserStatus {
//   USER_STATUS_UNSPECIFIED = 0;  // 未指定（零值）
//   USER_STATUS_ACTIVE      = 1;  // 正常
//   USER_STATUS_INACTIVE    = 2;  // 禁用
// }

// 更新操作中区分"未传"和"设为零值"：
// 使用 google.protobuf.FieldMask 或 optional 关键字
// syntax = "proto3";
// optional string nickname = 5;  // 显式可选，可区分未传和空字符串
```

## 小结

本文构建了一个**生产级 Go 微服务脚手架**，核心要点回顾：

1. **分层架构**：Handler(gRPC) -> Service(业务逻辑) -> Repository(数据访问) -> Model(数据模型)，职责分明，便于测试和替换。

2. **双协议暴露**：gRPC 用于服务间高效通信，HTTP Gateway 用于前端/外部调用，一份 Proto 定义生成两套接口。

3. **Cache-Aside 缓存策略**：读请求先查 Redis，未命中再查 MySQL 并回填缓存；写请求先更新 MySQL，再删除缓存。空值缓存防穿透。

4. **中间件链式调用**：Recovery -> Tracing -> Metrics -> Logging -> RateLimit -> Auth，顺序很重要——Recovery 必须在最外层确保 panic 不会导致进程崩溃。

5. **可观测性三支柱**：
   - **日志**：zap 结构化日志 + trace_id 关联
   - **指标**：Prometheus 请求计数 / 延迟直方图 / 缓存命中率 / 业务指标
   - **追踪**：trace_id 在 gRPC metadata 中跨服务传播

6. **配置管理**：Viper 支持 YAML 配置文件 + 环境变量覆盖 + `${VAR}` 占位符，实现 dev/staging/prod 多环境隔离。

7. **容器化开发**：Docker Compose 一键启动 MySQL + Redis + Prometheus + Grafana，Dockerfile 使用多阶段构建减小镜像体积。

8. **Makefile 自动化**：将所有日常操作封装为 make 目标，新人 clone 后只需 `make docker-up && make run` 即可启动完整环境。

掌握这套脚手架后，你可以在 30 分钟内搭建一个新的微服务，并且它天然具备监控、日志、缓存、认证等生产环境所需的基础能力。
